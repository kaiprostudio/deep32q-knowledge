# F01 營收因子與定價權分析 — 6290 良維

**分析日期**：2026-04-24
**資料期間**：2023-2025 (36個月)
**資料來源**：FinMind API (損益表、月營收)、公開財報

---

## 一、核心結論

**良維正處於營收結構性翻轉期**。2023年營收73.6億為近年低谷，但2024-2025年連續兩年創高，2025年營收突破101.6億，年增26.3%。毛利率自2023年低點22.8%攀升至2025年的31.4%，營業利益率從7.6%躍升至17.9%，定價權顯著增強。

**關鍵驅動因子**：數據中心電源連接器需求爆發、Apple供應鏈恢復、銅價轉嫁機制奏效。良維已從傳統消費電子連接器廠轉型為高毛利數據中心/伺服器電源解決方案供應商。

## 三、營收驅動因子分析

### 3.1 產品組合轉型 — 數據中心為核心動能

良維主要營收來源過去集中於消費性電子電源線/連接器（Apple供應鏈），但近年市況不佳。2023年營收73.6億為近年低點，主因筆電/手機需求低迷。

**2024-2025年的翻轉來自三大引擎：**

1. **數據中心/伺服器電源連接器**：AI伺服器建置潮帶動高功率電源線/匯流排需求。良維為Amazon、Google、Meta等CSP客戶的合格供應商，受惠於Capex擴張。
2. **Apple供應鏈回溫**：Apple Vision Pro及新款Mac/iPhone出貨恢復，傳統電源線業務不再衰退。
3. **電動車充電連接器**：長期布局開始貢獻營收，佔比仍低但成長快速。

### 3.2 定價權分析 — 明顯改善

**毛利率趨勢**：從2023Q1的13.4%低點，2024年提升至25%區間，2025年突破30%，Q4達34.8%歷史高點。毛利率的改善反映：

- **產品組合優化**：高毛利率的數據中心產品佔比提升，稀釋低毛利消費電子業務
- **銅價轉嫁能力**：良維主要原料為銅，2024-2025年銅價上漲期間，公司成功將成本轉嫁
- **規模經濟**：營收規模從73億增至102億，固定成本攤提效益顯現
- **降本增效**：2023年啟動的廠區調整（中國產能優化+東南亞擴產）逐步見效

**營業費用控制**：營業費用率從2023年的~18%降至2025年的~16%，規模效益明顯。

## 四、同業比較與競爭優勢

良維在台灣連接器同業中，毛利率改善幅度最為突出。相較於同業（如信邦、貿聯-KY等）毛利率多維持在25-28%區間，良維從23%躍升至31%，追趕速度最快。

**競爭優勢來源**：
- 深耕Apple供應鏈多年，品質認證門檻高
- 數據中心電源連接器技術門檻，需通過UL/CE等安規認證
- 客戶黏著度高，認證週期長（12-18個月）
- 生產基地分散（中國、越南、馬來西亞），地緣政治風險管理佳

## 六、結論與展望

良維2023-2025年完成了一次**產品組合升級**：從消費電子大宗品轉向數據中心高附加價值產品的結構性轉變。營收從73.6億成長至101.6億，毛利率從22.8%提升至31.4%，定價權顯著改善。

**展望2026年**：
- Q1月營收維持8.2-9.2億高檔，YoY仍有雙位數成長
- 2026年全年營收有機會突破120億
- 毛利率在產品組合持續優化下，有機會維持30-35%區間
- 主要風險為AI投資週期降溫及客戶集中度

**營收因子權重評估（2026年）**：
- 數據中心/AI需求：50%
- Apple供應鏈：25%
- 電動車/新能源：15%
- 其他（家電、工業）：10%
