# F03 合約負債與營收能見度分析 — 6290 良維

**分析日期**：2026-04-24
**資料期間**：2023-2025 (36個月)
**資料來源**：FinMind API (資產負債表)、公開財報

---

## 一、核心結論

**良維的合約負債規模極小，無法作為營收能見度的有效觀察指標。** 2023-2025年間，合約負債最高僅1,326萬（2023Q3），佔總資產比率從未超過0.17%。這反映良維的商業模式以即時/短期訂單交付為主，而非長約預收款模式。

**營收能見度只能透過客戶訂單動能與月營收趨勢間接判斷**。從2024-2026年月營收連續20個月正成長的趨勢來看，營收能見度實際上不低。

## 三、商業模式與合約負債關聯性

### 3.1 為什麼合約負債如此低？

良維的商業模式以**製造代工（OEM/ODM）**為主，主要產品為電源線組、連接器等電子零組件，交易特徵如下：

1. **短交期訂單模式**：客戶下單後數周至數月內交貨，非長期工程合約
2. **無大額預收款慣例**：連接器產業慣例以信用交易（30-90天）為主，鮮少預收
3. **庫存備料模式**：公司依據客戶預測備料，但預測不計入合約負債
4. **客戶認證週期長，但訂單週期短**：取得認證後，訂單按需求批次下達

### 3.2 產業對比

| 公司類型 | 合約負債佔比 | 代表 |
|:---|:---:|:---|
| 消費電子連接器(良維) | <0.5% | 6290 |
| 系統整合/工程合約 | 5-20% | 中鼎、亞翔等 |
| 半導體IP/設計服務 | 10-30% | 創意、智原 |
| SaaS/訂閱制 | 20-50% | - |
| 太陽能/風電 | 15-40% | - |

良維屬於合約負債極低的業態，這是商業模式自然所致，非公司經營問題。

## 六、結論

### 6.1 直接判斷
- **合約負債非營收能見度的有效指標**：規模極小（<0.2%），不適用
- **營收能見度須透過客戶關係與營運動能判斷**

### 6.2 間接判斷
- **營運動能強勁**：月營收連續20個月年成長，2025年全年營收年增26.3%
- **客戶結構轉佳**：從消費電子（季節性強）轉向數據中心（全年平穩），降低波動
- **認證時程正面**：2024-2025年新客戶認證結果，支撐2026年營收
- **無業績指引參考**：良維未提供正式的營收指引，能見度來自訂單能見度（通常3-6個月）

### 6.3 綜合營收能見度評級：**中高**

短期(1-3個月)：很高（月營收趨勢明確）
中期(3-6個月)：中高（客戶訂單能見度）
長期(6-12個月)：中（取決於AI Capex動能延續性）

**能見度來源**：月營收趨勢 > 客戶認證進度 > 法說會展望 > 合約負債（不適用）
