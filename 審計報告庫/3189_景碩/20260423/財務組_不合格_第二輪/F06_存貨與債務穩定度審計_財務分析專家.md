# F06: 存貨與債務穩定度審計 - finance-financial-analyst 審計報告

## 📋 審計題目
存貨金額變動趨勢為何？流動性資產是否足以覆蓋短期與長期債務？

## 🎯 專家直斷
截至2025年全年(2025/12/31)最新公開數據，景碩存貨自2023年初2.51B暴增至4.83B，增幅92.4%，遠高於同期營收成長(+67.2%)，存貨週轉天數由約47天延長至54天，顯示產能擴張過程中庫存水位堆積速度超越銷貨去化能力。速動比率由1.52降至1.17，流動比率由1.75降至1.44，雖現金12.28B仍足以覆蓋短期借款3.05B(4倍)，但長期借款11.20B合計總有息負債14.25B，現金覆蓋率僅86%，評級為【B(持平)】。

## 📊 診斷評級
B(持平) — 存貨堆疊速度顯著超越營收增長，存貨佔總資產比重由3.35%攀升至6.04%，製成品滯銷風險升溫。流動比率雖仍在1.4以上安全區，但下滑趨勢明確。速動資產(剔除存貨與預付款)對短期借款倍數為7.7倍，短期償債無虞；惟長期借款11.20B加上短期借款3.05B，合計14.25B逼近現金水位12.28B，考量每年資本支出約5-7B持續進行(ABF擴產)，現金流若未改善將觸及融資壓力。

## 🔍 答詢詳解

**1. 存貨趨勢縱向拆解：**
- 2023Q2: 2,508M → 2023Q4: 2,612M → 2024Q2: 3,036M → 2024Q4: 2,981M → 2025Q2: 4,261M → 2025Q4: 4,828M
- 近2.5年增量達2,320M，存貨佔總資產比從3.35%升至6.04%
- 存貨週轉率: FY2024約7.8x(天數47天) → FY2025約6.8x(天數54天)，週轉減慢15%
- 對比營收年增率: FY2025營收39.35B(+28.8%) vs. 存貨期末較年初+62%，存貨增速為營收增速2.15倍

**2. 償債壓力測試：**
- (現金12.28B+速動金融資產FVPL 1.41B+攤銷後成本2.64B) = 16.34B >> 短期借款3.05B → 短期流動性安全無虞
- 速動比 = (流動資產28.94B - 存貨4.83B - 預付款0.55B) / 流動負債20.11B = 23.56B/20.11B = 1.17
- 利息保障倍數: FY2025稅前淨利3.09B / 利息費用391M = 7.9倍，償息能力尚可
- 長期借款11.20B + 短期3.05B = 總有息負債14.25B，槓桿率(負債比)由FY2023的49.84%降至FY2025的48.27%，結構性改善但絕對金額隨擴產上升

**3. 跌價損失預警：**
- 存貨天數54天，對比載板同業(南電約45-50天、欣興約50-55天)處於行業中位以上
- 2025Q4存貨4.83B為歷史新高，若ABF載板景氣反轉，10%跌價損失即達483M，相當於FY2025全年稅後淨利2.72B的17.8%，衝擊不可忽視

## 💡 專家評析

景碩當前存貨堆積本質上反映的是ABF載板擴產週期的「非意圖性存貨」與「策略性備料」並存。以下為華爾街空頭視角阻力測試：

**極端情境模擬：** 若ABF載板需求2026下半年反轉，存貨週轉天數由54天惡化至75天(行業景氣下行期均值)，則需額外提列跌價損失約1.2-1.5B，將完全侵蝕營業利益。屆時速動比恐跌破1.0的流動性斷鏈警戒線。

**物理對策建議：**
1. 監控每季存貨週轉天數變化，若持續>60天且營收增速<15%，視為去庫存壓力明顯惡化訊號
2. 短期借款3.05B可透過現金全額清償，建議管理層降低短借以減少利息支出
3. 擴產資本支出規劃(235億)需嚴格匹配客戶長約(LTA)，避免產能開出後無訂單填充
