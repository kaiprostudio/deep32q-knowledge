# F03: 合約負債與營收能見度 - finance-financial-analyst 審計報告

## 📋 審計題目
合約負債與未來收入的變動關係為何？訂單轉化為實質營收的速度是否出現停滯？

## 🎯 專家直斷
截至2025年Q4最新公開數據，景碩合約負債維持在10.9億元水位(佔總資產1.36%)，自2023年Q4跳增至10.7億後已連續7季穩定在10-11億區間，無明顯增長動能。訂單能見度經測算約為10.1天(2025年Q4)，較2024年Q4的13.2天縮短23.5%，顯示訂單轉化速度加快但新訂單累積量並未同步增長。合約負債/營收比從2023年Q4的14.3%降至2025年Q4的10.1%，判定景碩的營收驅動模式為「即時接單出貨型」，非「預收款蓄積型」，訂單能見度短期，營運動能處於擴張期但合約負債領先指標未現爆發力。

## 📊 診斷評級
**評級：A（良）** — 合約負債穩定，訂單能見度維持在安全水位。景碩合約負債在2023年Q4從1.33億元跳升至10.7億元(受ABF載板景氣復甦驅動)，之後2024年全年維持在10.8-11.4億區間，2025年略降至10.3-10.9億。合約負債佔營收比從2024年Q4的13.7%降至2025年Q4的10.1%。合約負債年增率從2024年的+3.0%降至2025年的-1.4%。此數據組合顯示：合約負債作為營收領先指標並未提前反映2025年營收+28.9%的強勁成長，營收動能實際來自客戶的即時拉貨(非預付款)，與ABF載板缺貨環境下客戶傾向直接拉貨而非大額預付的產業慣性一致。

**1. 合約負債現狀核實（最近8季度）：**

2023Q4的合約負債跳升(從1.33億→10.7億)是ABF載板訂單強勁復甦的明確信號，此後營收從2023年Q4的74.9億穩步爬升至2025年Q4的108.1億(+44.4%)。但2024年Q1起合約負債便進入平台期，未再隨營收創高而增長。

**2. 訂單動能定量測算：**
- 合約負債變動率 vs 營收變動率彈性：
  - 2024年全年：合約負債+3.0% vs 營收+13.8% → 彈性係數0.22
  - 2025年全年：合約負債-1.4% vs 營收+28.9% → 彈性係數-0.05
- 合約負債/營收比從14.3%(2023Q4)持續下滑至10.1%(2025Q4)，顯示合約負債的營收覆蓋能力正在減弱。
- 判定：合約負債不具對營收的領先預測能力，營收動能需透過月營收年增率及法說會訂單展望來判斷。

**3. 訂單能見天數壓力測試（協議強制計算）：**
採用公式：`CurrentContractLiabilities / AvgRevenuePerDay`

以各季度最近12個月平均日營收計算：
- 2024Q4：1,104,108K / (30,534,979K/366天) = 1,104,108K / 83,429K/天 = 13.2天
- 2025Q1：1,100,293K / (31,992,537K/365天) = 1,100,293K / 87,651K/天 = 12.6天
- 2025Q2：1,025,485K / (33,801,491K/365天) = 1,025,485K / 92,607K/天 = 11.1天
- 2025Q3：1,042,729K / (36,389,075K/365天) = 1,042,729K / 99,696K/天 = 10.5天
- 2025Q4：1,088,434K / (39,351,096K/365天) = 1,088,434K / 107,811K/天 = 10.1天

訂單能見天數從2024年末的13.2天持續縮短至2025年末的10.1天(-23.5%)，主因為營收快速成長稀釋了合約負債的覆蓋力。10天能見度在半導體載板產業屬偏低水平，顯示景碩主要採即時接單出貨模式，而非累積大額預收款。

**4. 產業對位：**
ABF載板產業(欣興、南電、景碩)的合約負債普遍偏低，因客戶(IC設計公司、OSAT)的採購慣例以短期訂單為主，大額預付款較少見。相比之下，半導體設備商或晶圓代工廠的合約負債佔比(10-20%營收)遠高於載板產業。此為產業結構性特徵而非公司個體問題。

## 💡 專家評析

景碩的合約負債分析呈現一個有趣的悖論：營收創歷史新高(2025年Q4單季108.1億)，但合約負債卻反而從2023年Q4的10.7億微降至10.9億(幾乎持平)。這意味著公司的營收成長並非由「預收訂單蓄積→逐步釋放」的模式驅動，而是由「接單即出貨」的即時拉貨模式驅動。對於載板產業而言，此模式雖屬常態，但投資者需警惕：合約負債無法作為景碩的營收領先指標，必須依賴月營收公佈與法說會訂單展望來預測業績方向。法說會指引的「2026年優於2025年、營收毛利率同步成長」是更可靠的先行判斷。值得一提的是，合約負債在2025年Q2曾短暫降至10.25億(低谷)，隨後Q3/Q4回升至10.4-10.9億，顯示下半年訂單確有回溫。2026年若ASIC訂單(Google TPU、Meta等)正式放量出貨，合約負債可望突破前高站上12-15億區間，屆時將是營運動能進一步強化的明確訊號。當前評級維持A，但預期2026年下半年若合約負債突破12億可上調至S。
