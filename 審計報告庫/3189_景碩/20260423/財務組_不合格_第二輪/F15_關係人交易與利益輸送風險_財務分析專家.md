# F15: 關係人交易與利益輸送風險 - finance-financial-analyst 審計報告

## 📋 審計題目
是否存在異常大額的關係人交易(連屬公司、子公司)？其定價是否符合市場行情？是否存在利益輸送或虛增利潤之嫌？

## 🎯 專家直斷
截至2025年全年(2025/12/31)最新公開數據，景碩關係人交易規模極低，應收關係人帳款僅0.9M，其他應收關係人款項0元，佔總資產0.001%，可忽略不計。主要關係人為子公司晶碩光學(持股約57%)，其營運獨立且為公開上市公司(6491)，關聯交易具備市場透明度。唯一需關注的是2024年底處分大陸子公司蘇州統碩之交易與和碩(4938)同步取消，該交易涉及關聯方，但屬正常商業決策。治理風險等級判定為【A(良)】。

## 📊 診斷評級
A(良) — 關係人應收帳款餘額0.9M(2025Q4)、其他應收款0元，對比總資產79.97B、營收39.35B，佔比趨近於零。關係人交易主要為子公司晶碩光學的常規銷貨，晶碩本身為上市公司(6491)，財務資訊公開透明，具備外部市場定價錨。母公司對晶碩持股約57%，2025年法說會指出晶碩貢獻合併營收約18%，屬正常垂直整合範疇。無重大資金貸與、背書保證異常情形。

## 🔍 答詢詳解

**1. 交易規模實測：**
- 應收關係人帳款淨額逐季變動: 2023Q2: 1.5M → 2024Q2: 0.02M → 2025Q2: 0.01M → 2025Q4: 0.87M
- 其他應收關係人款項: 2024Q3: 7.0M → 2024Q4: 7.1M → 2025Q1: 1.1M → 2025Q2: 2.1M → 2025Q3: 4.8M → 2025Q4: 0
- 以上金額相對於總資產(~80B)及營收(~39B)可完全忽略，未有超過20%交易比例之高治理風險情形
- 關係人進銷貨方面，從法說會揭露晶碩貢獻合併營收18%，但這屬合併報表內子公司正常業務，非異常關聯交易

**2. 帳期異常偵測：**
- 關係人應收款週轉天數: 按0.9M餘額/年化關係人交易(假設約7B晶碩營收)推估<1天，顯著低於一般客戶週轉天數(~50天)
- 帳期極短反映集團內交易多以現金或即期結算，無資金占用風險
- 關係人應付款未在FinMind資產負債表中獨立揭露，側面證明金額極低

**3. 子公司利潤滲透與治理穿透：**
- 晶碩光學(6491)為獨立掛牌公司，FY2025營運穩定成長，無承接母公司不良資產或虛增利潤跡象
- 景碩持有晶碩57%，合併報表中已完全沖銷內部交易，無利潤藏匿空間
- 2024年12月景碩與和碩同步公告取消處分大陸子公司(蘇州統碩/世碩)，最初公告於2023年底。處分案取消原因是監管審批不確定性，決策合理，非異常利益輸送

**4. 內部人持股異動(必檢項)：**
- 董監持股比例: 2025年12月25.25% → 2026年3月28.47%，上升3.22ppts，趨勢正面

- 2024年9月經理人吳蔚文申報信託19張，金額僅2M，屬例行財務規劃，無異常拋售

## 💡 專家評析

從華爾街空頭視角審讀景碩的關係人交易，結論是：目前無明顯利益輸送跡象，但有兩個結構性觀察點：

**結構性治理評析：**
1. 景碩-晶碩的母子關係是典型的「控股母公司+金雞母子公司」結構。晶碩貢獻18%合併營收但隱形眼鏡業務與ABF載板本業完全無關，本質上是財務投資而非營運整合。若晶碩分拆或減資，母公司可能面臨一次性業外收益減少。
2. 和碩集團(4938)為景碩大股東背景，集團內部資源配置(如處分大陸廠區同步決策)顯示經營權高度集中。目前無異常，但投資人需追蹤大股東股權移轉情況。
3. 境外子公司架構尚待年報逐一穿透，尤其蘇州統碩的資產處置動向仍需關注。若景碩持續擴大海外子公司投資，建議每半年檢視關係人交易餘額變化。

**物理對策：**
- 設置關係人交易閾值監控: 若任一單項半年內關係人應收款>500M(佔營收>1.3%)需觸發深度審計
- 關注2025年年報之關係人附註對晶碩交易的逐筆披露
- 監控和碩集團(大股東層)的股權變動，若出現連續申報轉讓>1%持股，視為治理警訊
