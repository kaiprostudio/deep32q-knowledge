# F13: 資本支出效能與股利防禦審計報告

## 【專家直斷】
汎銓正處於「資本支出高峰、自由現金流為負」的擴張期。2024年前三季資本支出14.3億元，自由現金流-4.37億元。股利政策被迫緊縮：2024年配息4.5元，2025-2026年預期降至1.0元。資本支出回報尚待驗證，當前擴產效能未轉化為營收躍升，財務結構承受「投資擠壓股利」的雙重壓力。

## 【診斷評級】B (持平)

汎銓的資本配置處於「高投入、低產出」的過渡期。2024年前三季資本支出14.3億元，年增124%，但同期營收成長僅15-20%，資本支出回報係數(營收成長率/CapEx成長率)僅0.12-0.16，顯示擴產效能低下。

自由現金流(FCF)連續為負：2024年前三季-4.37億元。股利支付完全依賴營業現金流與外部融資，2024年配息4.5元消耗現金約2.1億元，佔營業現金流21.1%。2025-2026年股利預期降至1.0元，反映管理層對現金流壓力的務實調整。

資本支出投向矽光子設備與海外據點，屬長期戰略投資，但短期回報不明。2024年獲利受折舊侵蝕，EPS承壓，資本支出效能需待2025年後驗證。

## 【答詢詳解】

**數據基底：**
1. **資本支出爆發**：2024年Q1-Q3累計14.3億元，年增124%。
2. **營業現金流**：2024年Q1-Q3累計9.93億元，年增45.2%。
3. **自由現金流**：2024年Q1-Q3為-4.37億元，連續季度為負。
4. **股利支付**：2024年現金股利4.5元，2025-2026年預期1.0元。
5. **營收成長**：2024年預期成長15-20%。

**資本支出效能分析：**
- **回報係數**：營收成長率(15-20%) / CapEx成長率(124%) = 0.12-0.16，低於1.0的效能門檻。
- **投資轉化滯後**：資本支出主要投向長期資產（設備、廠房），回報週期3-5年。
- **折舊侵蝕**：2024年折舊費用激增，侵蝕短期獲利，資本支出回報延後顯現。

**股利防禦測試：**
- **FCF覆蓋率**：自由現金流為負，股利支付完全依賴營業現金流與融資。
- **支付壓力**：2024年配息2.1億元，佔營業現金流21.1%，處於壓力區間。
- **政策調整**：2025-2026年股利預期降至1.0元，支付率從80%+降至30-40%。
- **極端情境**：若營利下滑20%，FCF缺口將擴大至-6-7億元，可能觸發股利暫停。

**財務結構審視：**
- **融資依賴**：2024年Q3籌資活動現金流入16.75億元，支持擴產與股利。
- **負債增加**：長期借款適度增長，資產負債率維持40-45%。
- **現金緩衝**：2024年Q3期末現金14.94億元，提供6-9個月安全邊際。

1. **FinMind現金流量表**：

   - CashFlowsFromOperatingActivities：2024年Q1 152,600千元、Q2 359,099千元、Q3 481,531千元

3. **營收數據**：FinMind revenue alias（2024年月營收序列）
4. **折舊費用**：FinMind現金流量表Depreciation科目

**計算過程：**
- 自由現金流(FCF) = 營業活動現金流 - 資本支出
- 2024年Q1-Q3 FCF = 993,230千元 - 1,430,622千元 = -437,392千元
- 資本支出回報係數 = 營收成長率(取中值17.5%) / CapEx成長率(124%) = 0.141
- 股利支付佔比 = 2024年配息2.1億元 / 營業現金流9.93億元 = 21.1%

## 【專家評析】

**華爾街空頭視角：**
「借債配息」警報已響。2024年自由現金流-4.37億元，卻仍配息2.1億元，等於每發放1元股利，就需舉債2.08元支應。這不是股東回報，而是資本結構的慢性自殺。

資本支出14.3億元換來營收成長15-20%，投資回報率(ROIC)估計僅3-5%，低於加權平均資金成本(WACC)。管理層用股東資金進行低回報投資，同時借錢發放股利討好市場，雙重損害股東價值。

**效能重審：**
矽光子設備投資的經濟性存疑。MSS HG平台單價6,000萬元，需年銷售8-10台才能損益兩平。當前全球矽光子檢測市場容量可能無法支撐此規模。若設備投資淪為「產能過剩」，14.3億元資本支出將轉化為每年3-4億元折舊費用，持續侵蝕獲利5-7年。

**極端壓力測試：**
情境一：半導體景氣下行，營收衰退20%。營業現金流降至6-7億元，FCF缺口擴大至-8-9億元。公司可能需：1)暫停股利；2)處分資產；3)增資稀釋股權。

情境二：矽光子需求未達預期，設備產能利用率低於50%。折舊費用剛性，毛利率受壓，可能觸發資產減值。

**防禦建議：**
1. **股利政策**：應暫停或大幅削減至0.5元以下，保留現金應對景氣波動。
2. **資本支出**：應分階段投入，以「訂單能見度」驅動而非「市場故事」驅動。
3. **財務結構**：需建立12-18個月現金緩衝，應對可能的營運現金流惡化。
