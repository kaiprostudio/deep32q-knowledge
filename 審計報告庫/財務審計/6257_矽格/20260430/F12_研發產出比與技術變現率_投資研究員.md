# F12: 研發產出比與技術變現率 - 投資研究員 審計報告

## 📋 審計題目
> 近三年研發費用 (R&D) 投入與實際營收/獲利產出的關聯性為何？技術轉化效率是否正在改善？

## 🎯 專家直斷
- **核心結論**：矽格近三年研發費用率維持 2.31-2.74%，研發強度偏低且穩定；毛利率從 26% 攀升至 29%，但此改善來自規模效應與產品組合，非研發轉化驅動。研發槓桿（Revenue/R&D）從 36.5x 升至 43.3x，但增量貢獻持續下降，技術變現路徑不清晰。
- **風險評級**：中
- **數據時效**：根據截至 2025 年（114年度）最新公開數據

## 📊 診斷評級

**重點數據：**
- 研發費用：2023年 4.24億 → 2024年 4.35億 → 2025年 4.53億（YoY +2.7% / +3.9%）
- 營收：2023年 154.8億 → 2024年 182.2億 → 2025年 195.9億（YoY +17.7% / +7.5%）
- 研發費用率：2023年 2.74% → 2024年 2.39% → 2025年 2.31%（逐年微降）
- 毛利率：2023年 26% → 2024年 26% → 2025年 29%（YoY +3ppt）
- 營業利益率：2023年 20% → 2024年 17% → 2025年 20%（回彈）
- 研發產出倍數（Revenue/R&D）：2023年 36.5x → 2024年 41.9x → 2025年 43.3x

**診斷見解：**
研發費用率僅 2.3-2.7%，在半導體產業屬極低水準（IC設計業通常 10-20%、封測業 3-5%），反映矽格以「測試代工」為主的輕研發商業模式。毛利率從 26% 升至 29% 的改善幅度（+3ppt），與研發費用率微降（-0.43ppt）形成「反向背離」——即研發投入佔比下降的同時毛利率反而提升，暗示利潤改善動能來自規模經濟、良率優化或高階測試佔比提升，而非研發驅動的技術溢價。

## 🔍 答詢詳解

### 1. 研發槓桿分析 (R&D Leverage Audit)
- **核心重點**：2024年研發槓桿 = ΔRevenue / ΔR&D = (182.2-154.8)/(4.35-4.24) = 27.4億/0.11億 = 249x；2025年 = (195.9-182.2)/(4.53-4.35) = 13.7億/0.18億 = 76x。槓桿倍數從 249x 驟降至 76x，但仍為正。
- **深度見解**：2024年每增加1元研發費用帶來249元營收增量，2025年降至76元，邊際回報急劇衰減。營收增速從 17.7% 放緩至 7.5%，而研發費用增速從 2.7% 升至 3.9%——「投入增速 > 產出增速」的剪刀差正在形成，技術變現效率處於下降通道。

### 2. 技術路徑競爭力 (Tech Path Competitiveness)
- **核心重點**：矽格為封測代工（OSAT），研發主要投入先進製程測試（2nm封裝認證、AI/ASIC測試、矽光子測試），資本支出重點在中興三廠擴產。研發強度 2.3% 低於同業日月光（ASE）約 3-4%、矽品約 3.5%。
- **深度見解**：封測業的技術競爭力主要體現在「製程能力」而非「研發強度」。矽格的低研發費用率可能反映其以「執行力」取勝的策略，但長期而言，缺乏足夠的研發投入可能使其在先進封裝（如 CoWoS、扇出型封裝）的技術升級中落後於同業。

### 3. 專利質量與變現 (IP Value Monetization)
- **核心重點**：年報未提供具體專利數量與技術授權收入數據。研發費用中無「研發費用資本化」科目出現，暗示全部研發支出均為費用化處理。
- **深度見解**：全部研發費用化雖符合保守會計原則，但也意味著公司未將研發成果轉化為可量化之無形資產價值。缺乏專利授權收入（IP Licensing）表明技術變現路徑單一——僅透過「測試服務收費」實現，無額外的技術溢價收入來源。

## 💡 專家評析 (Skepticism Mandate)

- **空頭風險一：研發強度不足致技術斷層風險**
  矽格研發費用率僅 2.3%，遠低於封測同業 3-5% 的中位數。在先進封裝（CoWoS、矽光子）技術快速迭代的背景下，低研發投入可能使其錯失下一代測試技術窗口期。若主要客戶（如台積電生態系）轉單至研發更積極的同業，營收成長將面臨結構性壓力。2025年營收增速已從 17.7% 驟降至 7.5%，初步驗證此風險。

- **空頭風險二：研發轉化無效——毛利率改善非研發驅動**
  毛利率從 26% 升至 29% 的主因並非研發突破，而是規模效應（營收從 155億增至 196億，+26.6%）與可能的產品組合優化。研發費用率反而從 2.74% 降至 2.31%，形成「研發佔比下降、毛利率上升」的反常組合。一旦規模效應見頂，缺乏研發驅動的技術溢價將使毛利率面臨回落壓力。

- **空頭風險三：營收增速放緩與研發投入加速的剪刀差**
  2025年營收 YoY +7.5%（vs. 2024年 +17.7%），研發費用 YoY +3.9%（vs. 2024年 +2.7%）。營收增速腰斬的同時研發投入反向加速，研發槓桿從 249x 降至 76x。若此趨勢延續至 2026年，可能出現「研發費用率反彈但營收停滯」的惡性循環，侵蝕營業利益率。
