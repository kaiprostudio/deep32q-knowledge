# F13: 資本支出效能與股利防禦 — 投資研究員審計報告

## 📋 審計題目
> 資本支出 (CapEx) 的投入效能為何？自由現金流 (FCF) 是否足以支撐其股利政策？是否存在「借債配息」之虛耗現象？

## 🎯 專家直斷
- **核心結論**：2023–2025年 CapEx 年增 94%（2.99→5.80億），營收反降7%，FCF 連續三年為負且2025年惡化至-4.70億。股利覆蓋率=-13%，屬「借債配息」型態，資本配置效率評級 B（持平偏負）。
- **風險評級**：高
- **數據時效**：根據截至 2025 年第四季最新公開數據

## 📊 診斷評級
- **CapEx 成長 vs 營收成長（階差分析）**：
  - 2023→2024：CapEx +50.6%（2.99→4.50億），營收 +17.0% → 支出增速 > 收入增速
  - 2024→2025：CapEx +29.0%（4.50→5.80億），營收 -7.0% → 支出增加、收入反縮
- **FCF 覆蓋率（FCF / 總配息金額）**：
  - 112年（2023）：FCF=-0.80億 / 配息=0.92億 → **-87%**（無法覆蓋）
  - 113年（2024）：FCF=-0.98億 / 配息=0.36億 → **-274%**（無法覆蓋）
  - 114年（2025）：FCF=-4.70億 / 配息=0.63億 → **-748%**（無法覆蓋）
- **CapEx 回報係數（營收成長率 / CapEx 成長率）**：
  - 2024年：17.0% / 50.6% = **0.34**（<1，低效）
  - 2025年：-7.0% / 29.0% = **-0.24**（負向回報，極端低效）
- **資產支出率（CapEx / 營收）**：2023=16.8% → 2024=21.6% → 2025=29.9%（持續攀升，資本密集度加劇）

## 🔍 答詢詳解

2023–25年CapEx從2.99億增至5.80億（+94%），營收反降7%，FCF連續三年為負，2025年惡化至-4.70億。CapEx回報係數連續兩年低於1（2024=0.34、2025=-0.24），資產支出率從16.8%攀升至29.9%，資本密集度加劇。

FCF覆蓋率=-748%（-4.70/0.63），構成典型的「借債配息」——透過短期借款增加2.19億同時支撐資本支出與股東分紅。若A27產線驗證延遲或電子級硫酸價格承壓，FCF缺口將進一步擴大。

若2025淨利下滑20%（~0.31億），配息率將突破194%。公司唯有兩種選擇：（a）削減股利至0.28元/股以下，或（b）繼續舉債配息。現有股利政策缺乏防禦韌性。

A27廠年產能3萬噸，投資約12億TWD，2025年9月試產並送台積電驗證。若2026年驗證順利且產能利用率達70%+，營收有望恢復正成長；反之若驗證延遲或單價承壓，2026年FCF缺口可能超過-5億。

## 💡 專家評析 (Skepticism Mandate)
- **空頭風險一：A27 產線驗證失敗或延遲**——若台積電或其他大客戶驗證未通過，12億投資將沉沒為折舊費用。以 2025年淨利 0.39億計，每年折舊約 1.2億（按 10年直線），將使公司陷入常年虧損。此為最大單點風險。
- **空頭風險二：傳統石化硫酸業務結構性衰退**——2023–2025年營收從 17.8億降至 19.4億（含 A27 試產前全年），傳統業務已呈現結構性萎縮。若電子級硫酸無法在 2026年貢獻 30%+營收，公司將面臨「舊產能折舊 + 新產線折舊」雙重壓力，FCF 缺口可能突破 -5億。
- **空頭風險三：舉債配息的槓桿螺旋**——2025年短期借款增加 2.19億，利息支出侵蝕本已微薄的淨利。若未來利率維持高檔且營收無法跳升，利息覆蓋倍數將跌破安全線。
