## 📐 指標間關聯性

```
實質利率 ↓ → 美元 ↓ → 金價 ↑
通膨 ↑ → 實質利率 ↓ → 金價 ↑
地緣風險 ↑ → 避險需求 ↑ → 金價 ↑
央行購金 ↑ → 需求 ↑ → 金價 ↑
聯準會降息 → 實質利率 ↓ → 金價 ↑
```
