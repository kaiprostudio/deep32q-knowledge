## 摘要

2026 年黃豆市場經歷結構性轉變：**定價權從中國飼料需求轉移至美國生質柴油政策（RFS Set 2）**，壓榨利潤創 2.5 年高，豆油合約創歷史新高。此背景下，台股相關個股所受影響可分為三類：

1. **飼料/肉品加工廠**（大成、卜蜂）：黃豆是成本，**高黃豆價 = 成本壓力**
2. **油脂製造商**（福壽、泰山、福懋油）：黃豆是原料，**壓榨利潤擴大 = 利多**
3. **小型/專業廠**（台榮、興泰）：邊際影響，視業務組合而定

--- 

## 關鍵傳導鏈（重新釐清）

根據我們的結構性分析：

```

    ↓

    ↓

    ↓
CBOT 黃豆期貨跟漲（但漲幅小於豆油）
    ↓

    ↓
┌──────────────────┬─────────────────────┐
│   飼料廠           │   油脂廠              │

│  成本↑             │  壓榨利潤↑（理論上）    │
│  利潤↓             │  （實際上受台灣市場結構影響）│
└──────────────────┴─────────────────────┘
```

## 一、公司深度分析

### 1️⃣ 大成 (1210) — 台灣最大飼料/肉品廠

**基本資料**
| 項目 | 數據 |
|:---|---:|
| 股價（5/6） | NT$52.00 |
| 2026 Q1 營收 | 265.65 億（+4% YoY） |
| 2025 毛利率 | ~15% |
| 業務結構 | 飼料 54.4% / 食品 19.7% / 肉品 18.6% / 油脂 少量 |

**黃豆關聯度**：🟡🟡🟡🟡🟡（極高）

大成是台灣最大的飼料生產商，年消耗數百萬噸黃豆（進口）。飼料成本中，黃豆佔約 30-40%（另為玉米）。

**2026 成本情境**：

| 情境 | CBOT 黃豆 | 對大成影響 |
|:---|---:|:---|
| 基底（無戰爭） | $10.20-11.00 | ✅ 成本溫和，利潤正常 |
| **當前（戰爭 + RFS）** | **$11.50-12.00** | ⚠️ 成本升，公司稱庫存可撐 Q2 |
| 全面升級 | $12.50-14.00 | 🔴 下半年壓力大增，需調價 |

**公司因應策略**：
- 目前原料庫存可支撐到 Q2 底
- 下半年將視市場調漲飼料售價
- 新產能擴張中：水產飼料廠（年產值 30 億，已投產）、土雞加工廠（年產值 3 億）、全能生技廠 + 肉品分切廠（預計 2026/7 投產，合計 32 億）
- 越南畜牧場 + 美國冷凍食品持續擴張

**結論**：高黃豆價格對大成是**成本壓力**，需觀察公司轉嫁能力。若 CBOT 維持 $11.50-12.50，大成全年 EPS 可能被壓縮。買大成等於是**賭飼料價格能順利轉嫁**，不是純黃豆多單的替代。

---

### 2️⃣ 卜蜂 (1215) — 台灣第二大飼料/肉品廠

**基本資料**
| 項目 | 數據 |
|:---|---:|
| 股價（5/6） | NT$139.50 |
| 2026 Q1 累計營收 | 71.76 億（+7.16% YoY） |
| 2025 毛利率 | 21%+ |
| EPS 預估（統一證） | 6.92 元（法人區間 6.92-10.98） |

**黃豆關聯度**：🟡🟡🟡🟡🟡（極高）

卜蜂同樣是飼料/畜牧垂直整合大廠，黃豆為飼料核心原料之一。

**2026 壓力點**：
- 國際油價攀升推升運費 3-5 成
- 玉米+黃豆因生質燃料需求排擠飼料供給
- 公司目前約 3 個月低價庫存提供緩衝
- 若原料續漲，飼料價格可能上調

**因應策略**：
- 垂直整合：自有農場 + 直營通路
- 強化食品加工（高毛利產品）
- 成本轉嫁能力較大成強（市佔率較高 + 品牌力）

**結論**：卜蜂毛利結構較大成好（21% vs 15%），緩衝空間較大。但高黃豆價格仍是逆風。若看好黃豆繼續上漲，卜蜂不是適合的標的。

---

### 3️⃣ 福壽 (1219) — 百年油脂/飼料/食品廠

**基本資料**
| 項目 | 數據 |
|:---|---:|
| 股價（5/6） | NT$13.40 |
| 2026 Q1 累計營收 | 待確認（飼料穀物平穩向上預期） |
| 業務 | 食用油、飼料、肥料、寵物食品、肉品（洽富氣冷雞） |

**黃豆關聯度**：🟡🟡🟡🟡🟡（極高 — 垂直整合油脂 + 飼料）

福壽的特點：**既做油脂也做飼料**，黃豆價格上漲對兩邊影響互相抵銷。

- 油脂端：高黃豆油價有助維持榨油利潤
- 飼料端：高黃豆成本壓縮飼料利潤
- 1970 年代曾是美援黃豆溶劑提油工廠，技術底蘊深厚
- 洽富氣冷雞已供應星宇航空全艙等 + 部分華航，預計 2026 年轉虧為盈

**結論**：業務雙面性使福壽對黃豆價格波動較大成/卜蜂中性。公司預期飼料原料價格「平穩向上」，股價低（13.4 元）但流動性低，不適合大資金。

---

### 4️⃣ 泰山 (1218) — 老牌油脂/飲料/食品廠

**基本資料**
| 項目 | 數據 |
|:---|---:|
| 股價（5/6） | NT$18.85 |
| 2025 已轉虧為盈 | 經營權易主後 |
| 業務 | 油脂（沙拉油）、飲料、食品、食糧 |

**黃豆關聯度**：🟡🟡🟡🟡（高 — 油脂為核心業務之一）

泰山從油脂起家（1968 年沙拉油），黃豆主要來自美國和巴西。油脂仍為主要營收來源之一。

**2026 動向**：
- 經營權易主後聚焦三大成長動能：消費食品、食糧事業、國際布局
- 預計推出近百種新產品
- 深化商用食糧市場（連鎖餐飲 + 中央廚房）
- 北美零售通路、日本熊本市場拓展

**結論**：泰山油脂業務受惠高黃豆油價格，但非純油脂股（飲料食品佔比也大）。18.85 元價位接近淨值，若有整頓故事可能有趣，但不宜當作純黃豆題材。

---

### 5️⃣ 福懋油 (1225) — 純油脂/飼料

**基本資料**
| 項目 | 數據 |
|:---|---:|
| 股價（5/6） | NT$33.15 |
| 2026 Q1 營收 | 32.81 億（+1.88% YoY） |
| 業務 | 沙拉油、飼料、麵粉 |

**黃豆關聯度**：🟡🟡🟡🟡（高 — 油脂為主力）

福懋油以油脂業務為主，沙拉油（黃豆油）佔比高於飼料。在壓榨利潤擴大的環境下，**理論上是台股中最受惠的純油脂標的之一**。

**但需注意**：
- 台灣黃豆油市場是內需市場，與美國生質柴油市場無直接連動
- 福懋油的利潤取決於：**台灣沙拉油定價能力** vs **CBOT 黃豆進口成本**
- 若 CBOT 漲但國內沙拉油無法同比例漲價，壓榨利潤反而被壓縮

**結論**：潛在受惠標的，但需追蹤台灣沙拉油終端價格是否能跟漲。

## 二、投資框架：如何用黃豆研究來操作台股

### 核心矛盾

```
我們的研究方向：黃豆價格 → 偏多（RFS + 伊朗戰爭持續）
                ↓
台股誰受惠？問題比想像中複雜：
  油脂廠（福壽/泰山/福懋油）
    → 受惠：CBOT 黃豆油價格上漲
    → 但受限：台灣沙拉油是管制/寡佔市場，成本轉嫁不一定順暢

  飼料廠（大成/卜蜂）
    → 受害：進口黃豆成本上升
    → 但可轉嫁：飼料價格可調漲，只是有時間差

結論：沒有「純黃豆多頭概念股」這種東西。
      台股的飼料/油脂公司是成本轉嫁的博弈，不是期貨的替代。
```

### 真正該關注的

如果你認同我們的黃豆結構性多頭論述（RFS + 戰爭），在台股中：

| 標的 | 性質 | 最受益於 | 風險 |
|:---|:---|---:|:---|
| 福懋油 (1225) | 最純油脂股 | 壓榨利差擴大 | 台灣內需市場漲價受限 |
| 福壽 (1219) | 油脂+飼料 | 兩邊互補 | 流動性差、股價低 |
| 大成 (1210) | 飼料龍頭 | 飼料降價時 | 高黃豆=逆風，不建議 |
| 卜蜂 (1215) | 飼料/肉品 | 飼料降價時 | 同大成，但毛利較高 |

### 更好的策略

**如果你看多黃豆，直接做 CBOT 期貨，比買台股更純粹。**

台股這些公司：
- 股價受大盤氣氛影響更大
- 管理層的 hedging 策略未知
- 成本轉嫁有時間差（約 1-2 季）
- 台灣內需市場價格僵固

**唯一有意義的台股黃豆情境**：

```
情境：黃豆價格從 $12 跌回 $10-10.50
→ 大成/卜蜂飼料成本大幅下降
→ 雞豬價格若維持高檔（需求剛性）
→ 飼料廠利差暴增
→ 大成/卜蜂才有較明顯的獲利跳升
```

這與我們目前的劇本（黃豆偏多）**方向相反**。所以目前台股黃豆概念股沒有理想的做多機會。
