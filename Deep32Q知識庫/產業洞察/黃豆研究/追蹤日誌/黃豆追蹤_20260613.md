### 2️⃣ 巴西创紀錄1.77億噸豐收 — 全球供給壓力核心
- **最新數據**：
  - Conab預估2025/26巴西大豆產量1.771億噸（較去年+3.3%）
  - USDA 6月WASDE維持巴西產量高位
  - 巴西出口預估1.12億噸（+4.7% YoY）
  - Paranaguá報價較美灣低$0.80-$1.00/蒲式耳 → 巴西價格競爭力極強
- **風險**：若中國落實承諾大量採購美豆 → 巴西豆可能被排擠，但目前差距仍大
- **結論**：巴西豐收是全球黃豆價格最大的結構性壓力

### 3️⃣ 美伊和談突破 → 油價暴跌3.4% → 肥料成本壓力可能緩解
- **最新態勢**：
  - 川普6/12稱和談取得突破，和平協議最快6/14在瑞士簽署
  - Brent原油跌至$87.33（-3.4%），WTI跌至$84.88
  - 協議草案包含：美國解除伊朗石油制裁 + 伊朗重新開放荷莫茲海峽
- **對黃豆影響**：
  - 油價下跌 → 天然氣同步走低 → 氮肥（尿素）成本下降 → 種植成本壓力緩解
  - **但這對黃豆價格是間接利空**：成本支撐減弱
  - 長期看：若和談成功 → 全球通膨降溫 → Fed可能更早降息 → 美元走弱 → 有利大豆出口

### 4️⃣ 種植進度92% + 出苗79% — 超前五年均，天氣有利
- **最新USDA作物進度（6/8）**：
  - 種植92%（超前五年均80%+）
  - 出苗79%
  - 品質 G-E 65%（略低於預期但尚可）
- **天氣**：美國中西部天氣有利，未來一週降雨量175%正常值（愛荷華、密蘇里、伊利諾）
- **聖嬰確認**：NOAA 6/11發布El Niño Advisory，預期聖嬰將持續加強至2026-27年冬季
  - 美國：通常帶來涼爽濕潤夏季 → 有利大豆產量
  - 巴西/阿根廷：過量降雨風險 → 收割延遲 + 病害風險
- **結論**：短期供給壓力不減，$11.00支撐面臨考驗

### 5️⃣ 中美貿易：中國承諾25MMT vs 實際出口差距仍大 + $170億非大豆農產品協議
- **最新進展**：
  - 中國承諾25MMT 2026作物 + 每年25MMT至2028
  - 白宮5/18宣布中國額外承諾每年$170億非大豆農產品採購（至2028）
  - 但1-4月美對中出口仍年減48%
  - 本週出口銷售偏低，中國未列入主要買家
- **關鍵矛盾**：
  - 承諾25MMT vs 實際出口極低 → 差距巨大
  - 中國商務部稱協議為「初步性質」，無具體細節
  - Phase One教訓：承諾 vs 兌現差距
- **結論**：中國採購承諾是長期利多，但短期無法兌現，市場持懷疑態度
