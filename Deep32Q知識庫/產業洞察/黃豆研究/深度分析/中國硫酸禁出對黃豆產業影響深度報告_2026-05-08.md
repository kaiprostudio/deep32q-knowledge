## 1️⃣ 事件確認：中國硫酸出口封鎖

### 1.1 政策細節

| 項目 | 內容 |
|:---|:---|
| 宣布日期 | 2026/4/15（業界確認並新聞報導） |
| 生效日期 | 2026/5/1 |
| 終止日期 | 預計至 2026/12/31 |
| 管制範圍 | 普通工業硫酸、冶煉副產硫酸（銅、鋅冶煉副產品）|
| 豁免項目 | 電子級高純硫酸（需經特別行政批准）|
| 性質 | 實質性全面出口禁運（非配額）|
| 正式公告機關 | 中國商務部 / 海關總署 |

> 來源：ChemNet (2026-04-15)、GuanruChem (2026-04-16)、SCMP (2026-04-16)

### 1.2 政策背景

這不是中國第一次管制化肥出口，但**硫酸管制的殺傷力遠大於化肥管制**，因為：

1. **中國已先於 2026/3 全面凍結磷肥出口（DAP/MAP 至至少 8 月）**
2. **荷莫茲海峽受阻 → 中東硫磺無法運往中國 → 中國硫酸產能原料不足**
3. **中國國內化肥保供優先：春耕（4-6月）+ 秋播（9-10月）**

**關鍵因果鏈**：

```
荷莫茲海峽受阻 → 硫磺船運中斷
  → 中國硫磺進口 Q1 創 5 年新低
  → 中國硫酸產能受限
  → 優先保內需 → 禁止出口
```

> 來源：SCMP (2026-04-16)、WEF (2026-04)、WestPoint (2026-04)

### 1.3 禁令生效確認（5/8 更新）

禁令已正式生效一週，海關總署通告確認執行無誤：

- 海關總署通告：2026/5/1 起暫停普通工業硫酸及冶煉副產硫酸出口
- 電子級高純硫酸需特批（實質上也極難獲准）
- 無提供替代配額或補償方案
- 禁令前三個月（1-4月）出口量已設 70 萬噸配額，僅去年同期一半

> 來源：東方財富 (2026-05-02)、塗多多/新浪財經 (2026-05-07)

## 2️⃣ 為何硫酸對黃豆至關重要

### 2.1 化學路徑

```
硫磺(S) → SO₂ → SO₃ → H₂SO₄(硫酸)
                            ↓
磷礦石 + 硫酸 = 磷酸 → 磷肥(DAP/MAP)
                              [黃豆關鍵肥料]
```

### 2.2 黃豆對磷肥的依賴

- 每英畝 60 蒲式耳的黃豆產量，約吸收 **48 磅 P₂O₅**（磷酸鹽）
- 需施用約 **100 磅 DAP/英畝** 才能維持土壤養分平衡
- DAP 的生產：**硫酸約佔 DAP 生產成本的 30-40%**，用量約 **2.5-3 噸硫酸 / 噸 DAP**
- **硫酸約佔全球磷肥生產成本的 66%**（S&P Global，2026）

> 來源：MSU Extension、S&P Global (2026-04-15)、farmdoc Illinois (2026-05)

## 3️⃣ 三層傳導鏈

### 第一層：上游原料衝擊（荷莫茲海峽受阻）

- 中東硫磺出口中斷
- 中國硫磺 Q1 進口創 5 年新低
- 中國國內硫磺價格飆升至 **¥6,700/噸**（4/6 歷史高點，年初僅 ¥3,800）
- 中東硫磺 FOB **$700/噸**（年初 $100/噸 → 5 月 $700/噸，漲幅 **+600%**）
- 全球硫磺市場進入結構性短缺
- 港口硫磺庫存持續下降，未來 1-2 月預計仍為緊缺狀態

### 第二層：硫酸供應鏈斷裂（中國禁出）

- 中國占全球硫酸產量 **40%+**，亞洲出口總量 ~1,000 萬噸中占 **45%**，年出口 **460 萬噸**
- 5/1 起全面禁出 → 全球市場流失 **460 萬噸/年**
- 日本/韓國額外出口僅補 **50 萬噸**
- 全球淨缺口：約 **280 萬噸**
- 硫酸價格自年初已暴漲 **+130%**（年初 ~¥900/噸 → 5 月 ¥2,000-2,100/噸）
- 海關數據：今年前三個月出口量僅 **52.8 萬噸**，同比**腰斬**
- 國內冶煉製酸企業：禁令前外銷 400-600 元/噸，禁令後全部轉內銷 ¥2,000+/噸，**利潤暴增**

### 第三層：肥料與農業衝擊（核心）

全球磷肥市場同時面臨 **三重打擊**：

| 打擊 | 內容 |
|:---|:---|
| ① 中國磷肥出口凍結 | DAP/MAP 至 8 月，全球每年損失 800-1000 萬噸 |
| ② 硫酸禁出 | 非中國磷肥廠硫酸原料短缺，印度/巴西被迫減產 |
| ③ 硫磺價格飆漲 | 全球硫酸生產成本暴漲，DAP 價格維持高檔 |
| ④ **中國 DAP 生產虧損** | 國內生產成本 ¥4,800/噸，指導價僅 ¥4,250/噸，每噸虧損 ¥550+ |

> 註：第④項為關鍵新發現——中國化肥生產商在出口禁令與內銷限價間陷入雙重擠壓。國內 DAP 指導價雖為保護農民，但生產虧損已影響後續產能維持意願。

**最終結果**：

```
黃豆種植成本上升 + 產量可能下降 + 全球糧食安全壓力上升
```

## 6️⃣ 全球區域差異分析

### 高度暴露國家

| 國家 | 暴露原因 | 影響程度 |
|:---|:---|---:|
| 印度 | 全球最大磷肥進口國，高度依賴中國 DAP、硫酸 | 極高 |
| 印尼 | 鎳冶煉仰賴中國硫酸，NPI 產能受威脅 | 極高 |
| 巴西 | 全球最大黃豆出口國，磷肥依賴進口 | 極高 |
| 澳洲 | 銅冶煉仰賴硫酸進口 | 中高 |
| 智利 | 銅冶煉核心原料 | 中高 |
| 剛果(金) | 銅鈷冶煉 | 中高 |
| 墨西哥 | 化肥供應緊張 | 中 |

### 影響黃豆產業的關鍵路徑

```
中國硫酸禁出 → 全球硫酸短缺
  → 印度磷肥廠減產 → 印度進口替代需求暴增
    → 全球 DAP 價格被印度競價推高
  → 巴西磷肥進口受限 → 巴西黃豆種植成本飆升
    → 巴西擴產動能減弱 → 全球黃豆供應預期下調
  → 美國農民 DAP 成本上漲 34%
    → 玉米相比黃豆吸引力下降 → 黃豆種植面積上升
    → 但單位產量可能因施肥減少而下降
```

### 中國的邏輯

保護國內磷肥廠產能 → 不准硫酸出口 → 國內硫酸留在磷肥生產 → 增產磷肥 → 國內糧食增產（降低對美進口依賴）→ 同時減少化肥出口 → 海外競爭對手成本上升

## 7️⃣ 對黃豆種植的具體影響

### 種植面積效應

| 因子 | 對黃豆面積影響 | 機制 |
|:---|---|:---|
| 磷肥成本上升 | ↑ 正面 | 黃豆需肥量遠低於玉米，相對優勢凸顯 |
| 氮肥暴漲 | ↑ 正面 | 黃豆為豆科自固氮，玉米氮肥成本大幅上升 |
| 黃豆相對利潤 | → 中性偏正 | 預期收益減肥料成本：黃豆+$70/英畝、玉米+$73/英畝 |

**結論**：黃豆面積看增，多個來源確認 2026 美國黃豆種植面積增加。

### 單位產量效應

- 肥料價格每漲 1% → 黃豆面積減 2.616%（但單位產量反增 0.819%，效率極大化）
- 2026/4 全美農民調查：**70% 農民買不起所需肥料**
- 無法充分施肥的農場 → 單位產量下降

**綜合判斷**：高機率看到總產量不如預期。

### CBOT 黃豆價格路徑

**核心觀點**：無論美伊談判結果如何，硫酸禁出的影響是獨立的、持續到 2026 年底的結構性因素。

| 情境 | 黃豆價格 | 邏輯 |
|:---|---:|:---|
| 美伊 MoU 簽署 | $11.50-12.50 | 原油跌 + 和緩，但硫酸缺口 280 萬噸獨立存在 |
| 美伊協議破裂 | $12.50-14.00 | 原油飆 + 硫磺續缺 + 全球恐慌採購 |

## 8️⃣ 情景分析

### 情景 A：樂觀（20% 機率）

- 美伊全面協議 + 荷莫茲重開 → 硫磺恢復 → 硫酸生產正常化
- 中國提前解禁（早於 12/31）
- ADNOC/Ghasha 提前產能
- 黃豆價格：**$10.50-11.50**，回落至成本支撐線

### 情景 B：基準（55% 機率）

- MoU 短期停火 + 部分海運恢復 → 硫磺供應中低度恢復
- 中國維持出口禁令至年底 → 硫酸缺口 200-280 萬噸
- 印度競價推高 DAP → DAP $750-800/噸
- 黃豆價格：**$11.50-12.50**（成本推升 + 面積博弈）

### 情景 C：悲觀（25% 機率）

- 美伊全面開戰 + 荷莫茲長期封鎖 → 硫磺供應中斷數月
- 中國硫酸禁令擴大至電子級 → 硫酸缺口 >300 萬噸
- 印度/巴西/印尼磷肥廠減產 → DAP $800-900/噸+
- 黃豆價格：**$12.50-14.00**（供應衝擊 + 恐慌定價）

## 9️⃣ 結論與策略啟示

### 核心總結

1. **硫酸禁出非短期事件**：至少持續到 2026 年底，且與美伊戰事**獨立**影響肥料供應鏈
2. **三層疊加效應**：硫磺短缺 x 硫酸禁出 x 磷肥凍結 = 化肥市場完美風暴
3. **對黃豆多空交錯**：
   - **多**：成本推升 + 相對優勢（vs 玉米）→ 價格底部 $11.50
   - **空**：若全球經濟衰退壓制需求，成本支撐未必守住
4. **無論美伊結果，硫酸缺口不會消失**
5. **時間滯後**：2026 年已有預購的農民影響有限 → **2027 年將為全面衝擊年**
