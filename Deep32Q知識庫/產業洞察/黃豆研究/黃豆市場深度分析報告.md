## 三、天氣與種植進度

### 3.1 美國中西部

- **種植季**：4-5 月，目前進度正常偏快
- **天氣風險**：中西部春季降雨偏多，可能延遲種植
- **關鍵觀察**：種植窗口期天氣將決定最終面積

### 3.2 巴西

- **收割進度**：2026 年收割已接近尾聲，產量創紀錄
- **二季作物**：玉米種植面積可能因黃豆獲利更高而減少
- **風險**：若收割期遇雨，可能影響品質

### 3.3 阿根廷

- **乾旱風險**：拉尼娜現象可能影響產量
- **關鍵時期**：12-2 月開花結莢期是關鍵

## 四、政策與貿易戰影響

### 4.1 美中貿易戰

| 措施 | 影響 |
|------|------|
| 美國對中國關稅 145% | 黃豆出口受阻 |
| 中國報復性關稅 | 迫使需求轉向巴西 |
| 農業補貼 | 美國政府提供補貼緩衝 |

### 4.2 中國需求轉移

- **巴西成為首選**：中國從巴西進口黃豆比例持續上升
- **供應鏈多元化**：中國減少對美國依賴，增加阿根廷、烏拉圭等來源
- **匯率影響**：美元走強不利美國出口，人民幣走勢影響中國進口成本

### 4.3 貿易流向重組

```
傳統：美國 → 中國（最大買賣）
現在：巴西 → 中國（首選）+ 美國 → 中國（減少）+ 阿根廷 → 中國（增加）
```

## 六、投資/交易建議

### 6.1 短期（1-3 個月）

- **方向**：中性偏多，震盪格局
- **策略**：1150 以下逢低佈局，1200 以上減碼
- **風險**：巴西收割結束後的供給壓力

### 6.2 中期（3-6 個月）

- **關鍵事件**：美國種植季天氣、中國進口數據
- **方向**：若天氣正常，偏空（巴西豐收 + 美國擴種）
- **策略**：1200 以上可佈局空單

### 6.3 長期（6-12 個月）

- **結構性因素**：巴西持續擴產、中美貿易戰常態化
- **方向**：偏空，但需關注天氣黑天鵝
- **關鍵觀察**：2027 年巴西能否再破紀錄

## 七、結論

1. **供給充裕**：巴西創紀錄 + 美國擴種，全球供給偏鬆
2. **需求穩定**：中國進口需求維持高檔，但轉向巴西
3. **價格展望**：短期震盪，中期偏空，長期需關注天氣
4. **最大變數**：美國種植季天氣 + 中國需求變化
