## 一、肥料成本對黃豆種植的影響

### 1.1 黃豆種植成本結構

| 項目 | 佔比 | 每英畝成本 |
|------|------|----------|
| 種子 | 8-12% | $60-80 |
| **肥料** | **15-20%** | **$80-120** |
| 農藥 | 8-12% | $50-80 |
| 人工 | 10-15% | $60-100 |
| 機械/設備 | 15-20% | $100-130 |
| 土地租金 | 25-30% | $150-200 |
| **總成本** | 100% | **$550-750** |

### 1.2 肥料價格敏感度

| 肥料漲幅 | 總成本影響 | 盈虧平衡價影響 |

### 1.3 盈虧平衡價格

- **當前盈虧平衡價**：$10.40-13.54/蒲式耳
- **當前 CBOT 報價**：~$11.66/蒲式耳
- **結論**：多數農民處於**微利或虧損**狀態

## 三、中東戰事影響

### 3.1 荷莫茲海峽風險

- 全球 **20% 石油運輸**通過荷莫茲海峽
- 中東尿素出口高度依賴海運

## 四、尿素對黃豆的傳導機制

### 4.1 傳導鏈路

```
肥料成本↑ → 種植成本↑ → 種植意願↓ → 面積↓ → 供給↓ → 黃豆期貨↑
         ↓
       玉米更貴 → 農民轉種黃豆 → 黃豆供給↑ → 價格↓
```

### 4.2 量化模型

- **傳導係數 α = 0.4-0.6**
- 尿素每升 10% → 黃豆期貨漲 4-6%
- 歷史相關性：尿素 vs CBOT 黃豆 +0.72~0.81
- 傳導滯後：6-12 個月

### 4.3 黃豆固氮緩衝

- 黃豆有固氮能力，肥料漲價衝擊比玉米小
- 但間接效應（轉種）可能增加黃豆供給

## 六、結論

**黃豆期貨處於「供需偏鬆 + 成本支撐」的拉鋸格局。**

- **下檔支撐**：肥料成本（盈虧平衡 $1,040）+ 中國需求
- **上檔壓力**：巴西豐收 + 美國擴種 + 貿易戰
- **突破條件**：尿素 $700+ 或美國天氣災害
- **操作建議**：區間操作為主，1,100 以下逢低佈局，1,200 以上獲利了結
