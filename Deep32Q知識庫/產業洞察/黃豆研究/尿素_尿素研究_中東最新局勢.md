# 尿素研究：中東最新局勢（2026-04-27）

## 中東戰事現況

### 荷莫茲海峽
- 全球 20% 石油運輸通過此海峽
- 中東尿素出口高度依賴海運
- 若封鎖 → 中東尿素出口中斷 → 全球供給缺口 ~5-8%

### 主要中東尿素生產國

| 國家 | 全球產量佔比 | 主要出口市場 |
|------|------------|------------|
| 沙烏地阿拉伯 | ~8% | 印度、東南亞 |
| 卡達 | ~5% | 印度、東南亞 |
| 阿曼 | ~3% | 印度、非洲 |
| **合計** | **~16%** | 亞洲為主 |

### 戰事情境分析

| 情景 | 機率 | 尿素影響 | 黃豆影響 |
|------|------|---------|---------|
| 低度衝突（現狀） | 60% | +5-10% | +2-4% |
| 海峽短暫封鎖 | 25% | +30-50% | +12-20% |
| 全面戰爭 | 10% | +80-120% | +25-40% |
| 停火協議 | 5% | -10-15% | -3-5% |

### 傳導鏈
```
中東戰事升級 → 荷莫茲海峽風險 → 中東尿素出口中斷 → 全球供給缺口 → 尿素價格飆漲 → 種植成本↑ → 黃豆供給↓ → 期貨↑
```

## 對黃豆的綜合影響

- 當前尿素 $480-580/噸，未達 $700 閾值
- 若戰事升級導致尿素突破 $700 → 黃豆有 8-15% 上漲空間
- 最大變數：荷莫茲海峽通航狀態
