## 二、NVIDIA 液冷供應鏈層層拆解

### 2.1 GB300 架構（當前主力）

```
NVIDIA Grace-Blackwell B300 GPU (TDP 1,400W)
    │
    ├── 液態金屬介面材料（73 W/m·K）→ 富世達(6841)
    │       ↓
    ├── 獨立小型冷板 × 6/計算托盤 → 奇鋐(3017) 40~50%市佔
    │       ├── 鍍鎳防腐處理
    │       └── 雙相D2C冷卻（2026-2027大規模採用）
    │       ↓
    ├── 分歧管（Manifold）→ 奇鋐、雙鴻(3324)
    │       ↓
    ├── UQD v2 快速接頭 → 雙鴻(3324)
    │       ↓
    └── CDU（列間冷卻分配器）→ 台達電(2308)、雙鴻(3324)
            ↓
        冷卻塔/乾冷器 → 資料中心基礎設施
```

```
VR200 GPU (TDP 2,300W) + Rubin GPU (TDP 1,800W)

    │       └── 2027年量產 Rubin Ultra (~3,000W)

    │       ↓
    └── 無風扇模組化設計 → 晟銘電(3013) Sidecar技術
```

## 五、產業標準與生態系成熟度

- **UQD v2 標準化**：NVIDIA推動快速接頭標準，零組件規模化
- **OCP MGX規範**：72 psig最大操作壓力、217 psig最小爆破壓力
- **溫水冷卻趨勢**：45~65°C冷卻液回流，降低PUE至1.02~1.15
- **系統級價值提升**：從單一冷板→CDU+分歧管+泵浦的完整系統方案
