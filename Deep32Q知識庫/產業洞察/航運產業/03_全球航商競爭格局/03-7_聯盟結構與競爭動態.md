# 貨櫃航運聯盟結構與競爭動態（2025-2026）

## 聯盟格局重大重組
2025年是貨櫃航運聯盟歷史上的分水嶺。經過十餘年的聯盟競爭後，2025年初發生了根本性重組：

### 解體的舊聯盟
- **2M Alliance**（馬士基 + MSC）：於2025年1月正式結束，合作長達十年。雙方選擇各自獨立發展戰略
- **THE Alliance**（赫伯羅特 + 長榮 + HMM + ONE）：2025年1月底赫伯羅特退出，剩餘成員重新組建

### 現行三大聯盟/合作架構

#### 1. Ocean Alliance（海洋聯盟）— 最大聯盟
- **成員**：CMA CGM、中遠集團（COSCO）、OOCL（东方海外）、長榮（Evergreen）
- **總運力**：約500萬TEU，擁有約390艘貨櫃船
- **市場佔有率**：2025年約29%（部分分析顯示高達28.4%-31%）
- **總容量**：考慮訂單簿後超過950萬TEU，為所有聯盟中最大
- **協議期限**：已延長至2032年3月，是唯一保持原始結構的主要聯盟
- **優勢航線**：跨太平洋和亞歐航線，特別是亞太-北歐航線峰值可達41%市場份額

#### 2. Gemini Cooperation（雙子星合作）— Maersk + Hapag-Lloyd
- **成立時間**：2025年2月正式運營（赫伯羅特於2025年1月底退出THE Alliance後加入）
- **規模**：約340艘船舶，總運力約370萬TEU（Maersk佔60%，Hapag-Lloyd佔40%）
- **市場佔有率**：2025年約21.5%-22%
- **網絡覆蓋**：34條共享主航線 + 29條穿梭航線，12個核心樞紐的樞輻模式
- **目標**：關鍵航線準點率超過90%（顯著高於行業歷史平均）
- **趨勢**：部分主要航線份額呈下降趨勢（如亞太-美國西海岸從15%降至13%，亞太-北美東岸從20%降至17%），原因是其他聯盟擴大部署

#### 3. Premier Alliance（卓越聯盟）— ONE + 長榮 + HMM
- **前身**：THE Alliance（赫伯羅特退出後重新命名）
- **成員**：ONE（海洋網絡快運）、長榮（Evergreen Line）、HMM（韓進海運）
- **性質**：較小的聯盟組合，透過合作維持跨洋航線競爭力

#### MSC（獨立運營）
- 脫離2M Alliance後選擇完全獨立運營
- 提供直接港口對港口的廣泛服務網絡（不依賴聯盟）
- 以船隊規模優勢（720萬+ TEU，全球最大）和靈活策略維持市場領導地位

## 市場集中度分析
- **前六大航商合計**：MSC (21.4%) + Maersk (~13-14%) + CMA CGM (12.7%) + COSCO (10.5-10.9%) + Hapag-Lloyd (~8%) + ONE (~6%) = 約75-78%的全球貨櫃運力
- **聯盟控制**：三大聯盟/合作架構合計控制約70%以上的全球貨櫃運力，市場集中度極高
- **趨勢**：市場持續向大型聯盟集中，中小型航商生存空間被壓縮

## 2026年關鍵趨勢
1. **地緣政治影響**：紅海危機和荷姆茲海峽關閉正在重塑聯盟的航線策略和市場份額
2. **過載運力風險**：大量新造船訂單（特別是超大型船）可能在2027-2029年集中交船，導致運力過剩
3. **綠色轉型壓力**：各聯盟和獨立航商都在投資甲醇/LNG雙燃料船舶，環保合規成本上升
4. **服務可靠性競爭**：Gemini Cooperation的90%準點率目標正在推動行業整體服務標準提升
