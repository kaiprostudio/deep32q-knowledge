# 散裝航運獲利模型

## 📊 TCE（Time Charter Equivalent）計算方法

TCE 是散裝航運業衡量船舶每日營收的核心指標，用於跨航次、船型、租約方式比較獲利能力。

### 計算公式
```
TCE = (總運費收入 – 航次成本) ÷ 航次天數
```

### 成本組成
| 項目 | 佔比 | 說明 |
|------|------|------|
| **燃油成本** | 30–50% | 最大變數成本，受油價、航線、船齡影響 |
| **港口費用** | ~21% | 裝卸港費、引水費、繫泊費等 |
| **運河通行費** | 可觀 | 蘇伊士 $30–70 萬、巴拿馬 $20–50 萬 |
| **經紀佣金** | 1.25% | 成交額的 1.25%，部分航線可議價 |
| **其他** | 剩餘 | 保險、通訊、雜費等 |

### 損益平衡日租金
```
損益平衡 TCE = 每日營運成本 + 每日資本成本
```

## 🔍 影響散裝獲利的關鍵變數

### 利多因素（2026）
1. **Simandou 鐵礦項目**：幾內亞大型鐵礦開採，長距離運輸至中國推升 Capesize 需求
2. **澳洲/巴西出口強勁**：鐵礦砂、煤炭出口量創高
3. **船隊增長放緩**：Capesize 淨增長僅 2.2%，連續五年低於平均
4. **航程拉長**：幾內亞→中國航線 + 港口等待時間增加有效需求

### 風險因素
1. **Panamax 新船交付**：2026 年預計交付 15–16 百萬 DWT（近年約 10 百萬），供應壓力增大
2. **中國房地產放緩**：鐵礦進口增速可能放緩
3. **煤炭貿易萎縮**：Panamax 主要貨物之一，受能源轉型影響
4. **EEXI/CII 降速**：老船需降速 1–2 節，影響營運效率

## 📊 典型航次獲利範例（Capesize，2026 Q1）

### 巴西→中國鐵礦航次
- **運費收入**：約 $45–50 萬/航次（按當前即期費率）
- **燃油成本**：約 $18–22 萬（VLSFO ~$963/MT，耗油約 150–200 MT）
- **港口費用**：約 $8–12 萬（巴西裝港 + 中國卸港）
- **運河費**：約 $0（繞好望角，不經巴拿馬）
- **航次天數**：約 35–40 天（巴西→中國）
- **TCE**：約 $25,000–$30,000/天
- **淨利**：約 $15,000–$20,000/天（扣除營運成本後）
