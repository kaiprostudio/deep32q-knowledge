## 🥇 第一梯隊：核心受惠股

### 1. 昇達科（3491）— LEO 純度最高、確定性與成長性雙冠

```
上市櫃：上櫃
供應鏈層級：射頻/模組（濾波器/波導管/OISL 光星間鏈路）
LEO 營收佔比：高（LEO 主要成長動能）
客戶涵蓋：Starlink + Kuiper + Blue Origin + AST SpaceMobile
```

**核心數據**
| 指標 | 數值 | 年增 |
|:---|:---:|:---:|
| 2026 Q1 營收 | **10.2 億** | **+64.5%** |
| LEO 產品營收 | 季增 **40%** | 年增 **131%** |
| 毛利率 | **58%** | 持續提升 |
| 在手訂單 | **18 億** | — |

**競爭護城河**
- 全球極少數同時打入 Starlink、Kuiper、Blue Origin、AST SpaceMobile 四家 LEO 營運商的台廠
- OISL（Optical Inter-Satellite Link）雷射星間鏈路組件已量產→高技術壁壘
- 衛星酬載（太空）+ 地面站 + 終端（地面）全覆蓋
- 毛利率 58% 反映產品技術領先

**關鍵催化劑**
- SpaceX IPO 預計 2026/6 → 擴大資本規模、激勵供應鏈
- Starlink V3 Gen3 預計 2026 H2 Starship 首發 → 訂單跳增
- 衛星汰換週期 2027-28 → 雙重需求（新增+汰換）
- AST SpaceMobile 美商用 2026 年底 → 新客戶放量

**主要風險**
- 客戶集中度：雖涵蓋多家營運商，仍以 SpaceX 為大宗
- SpaceX IPO 若延遲或失敗將影響市場信心
- 技術路線變遷（頻段標準未完全確立）

### 2. 華通（2313）— 全球衛星 HDI 板絕對霸主

```
上市櫃：上市
供應鏈層級：PCB/材料（衛星 HDI 板）
LEO 營收佔比：~21%（2025）
客戶涵蓋：Starlink（主要）+ Kuiper（出貨中）
```

**核心數據**
| 指標 | 數值 | 備註 |
|:---|:---:|:---|
| 2025 全年 LEO 板營收 | **>153 億** | 佔總營收 ~21% |
| 2026 LEO 板目標 | **挑戰 200 億** | 年增 ~30% |
| 全球衛星 HDI 板市佔 | **>90%** | 近乎壟斷 |
| 天上板佔比 | 33%（持續提升） | 技術含金量更高 |
| 泰國廠 | 量產中 | 2026 產能擴大 |

**競爭護城河**
- 全球衛星 HDI 板市佔 >90%，近乎壟斷——Starlink 沒有第二選擇
- 天上板（太空級）技術門檻極高：抗輻射、熱管理、可靠度認證
- SpaceX + Kuiper 雙客戶、第二個 LEO 客戶出貨放量中
- 泰國廠新增產能支持 2026 年成長

**關鍵催化劑**
- Kuiper 全面商用 2026 年底（26 國）→ LEO 板需求跳增
- Starlink V3 Gen3 單顆衛星容量 20 倍於 F9 → 電路板設計升級
- 第二顆 LEO 客戶確認放量 → 降低 SpaceX 集中風險
- 衛星汰換週期 2027-28 → 既有衛星回購訂單

**主要風險**
- 客戶集中度極高（SpaceX+Kuiper 兩家）
- 若 Kuiper 出貨不如預期將影響成長
- 地緣政治：若美中衝突升級影響供應鏈運作

### 3. 台光電（2383）— 高頻 CCL 隱形冠軍

```
上市櫃：上市
供應鏈層級：材料（高頻銅箔基板 CCL）
LEO 營收佔比：中（主要成長動能之一）
客戶涵蓋：Starlink + Kuiper（間接供應）
```

**核心數據**
| 指標 | 數值 | 備註 |
|:---|:---:|:---|
| 全球高頻 CCL 市佔 | **~70%** | 領導地位 |
| 產能利用率 | 全產全銷 | 2026 延續滿載 |
| LEO 應用 | 衛星酬載板 + 地面站 | 雙渠道 |

**競爭護城河**
- 全球高頻 CCL 市佔 ~70%，規模與技術領先難以撼動
- LEO 衛星對高頻材料需求（Ka/Ku 頻段）遠高於傳統通訊設備
- 華通 HDI 板的上游關鍵材料供應商，形成「台光電→華通→Starlink」供應鏈
- 全產全銷反映市場供不應求

**關鍵催化劑**
- SpaceX + Kuiper 雙引擎持續放量
- 華通 LEO 板目標 200 億 → 上游材料需求同步成長
- 高頻材料替代空間有限，全球僅少數供應商

**主要風險**
- 原物料價格波動（銅箔、玻纖布）
- 中國競爭者追趕高頻 CCL
- LEO 佔整體營收比重仍有限

## 第二梯隊：重要參與者

### 4. 啟碁（6285）— 台廠 LEO 產值最大

```
上市櫃：上市
供應鏈層級：地面設備（ODU 終端、平板天線、地面站）
LEO 營收佔比：~10%（但絕對值最大）
客戶涵蓋：Starlink + Kuiper
```

**核心數據**
| 指標 | 數值 | 備註 |
|:---|:---:|:---|
| LEO 營收 | **~120 億** | 台廠 LEO 產值最大 |
| 2026-2027 訂單 | 法人預估 **倍增** | 成長動能強勁 |
| 佔整體營收 | ~10%（提升中） | 目標 LEO 佔比 30-40% |
| 產品線 | ODU + 平板天線 + 地面站 | 完整覆蓋 |

**競爭優勢**
- 台廠中 LEO 相關營收絕對值最大（~120 億），規模效應
- 產品線完整：終端設備 + 地面站 + 天線
- 客戶涵蓋 Starlink 與 Kuiper 兩大系統
- 若 LEO 佔比目標 30-40%→營收結構質變

**關鍵催化劑**
- Kuiper 全面商用 2026 年底 → 終端需求爆量
- 法人預估 2026-2027 訂單倍增
- LEO 佔比從 10% 提升至 30-40% 的估值重塑

### 5. 穩懋（3105）— 全球 GaAs 代工龍頭

```
上市櫃：上櫃
供應鏈層級：半導體晶圓（GaAs/GaN 射頻 PA 代工）
LEO 營收佔比：基礎建設 30%+
客戶涵蓋：Starlink + Kuiper
```

**核心數據**
| 指標 | 數值 | 備註 |
|:---|:---:|:---|
| GaAs 代工 | 全球市佔第一 | 產能規模最大 |
| 衛星佔基礎建設營收 | **30%+** | 比重顯著 |
| 頻段覆蓋 | Ka/Ku/V/E/W | 全面佈局 |
| 直連手機 | T-Mobile、AT&T、Verizon | 終端射頻需求 |

**競爭優勢**
- 全球 GaAs/GaN 射頻 PA 代工龍頭
- LEO 衛星從地面到太空都需要射頻 PA（多頻段、多規格）
- V/E/W 頻段開發中，卡位下一代 LEO 通訊技術
- D2C（直連手機）終端射頻需求 Starlink Mobile 日增 ~18,000 用戶

**關鍵催化劑**
- V3 Gen3 衛星需求更高頻段 → 射頻含量提升
- Starlink Mobile 用戶爆發 → 終端射頻晶片需求倍增

### 6. 台揚科技（2314）— OneWeb 專屬供應鏈

```
上市櫃：上市
供應鏈層級：地面設備（衛星收發模組、ODU、追蹤天線）
LEO 營收佔比：高（主力成長動能）
客戶涵蓋：OneWeb + Telesat + Viasat
```

**核心數據**
| 指標 | 數值 | 備註 |
|:---|:---:|:---|
| 主要客戶 | OneWeb（主力）| Eutelsat Group |
| 新訂單 | OneWeb Gen2 340 顆（Airbus 2026/1 簽約）| 交貨 2026 底 |
| 出貨週期 | **2026 H2 交貨高峰** | 持續 **3 年** |
| 產品 | Ku/L 頻段收發模組、自動雙軸追蹤 | 完整終端 |

**競爭優勢**

**關鍵催化劑**
- OneWeb Gen2 發射 2026 底 → 地面終端訂單確認

## 第三梯隊：成長潛力股

### 7. 聯發科（2454）— NTN 數據機晶片供應商

```
代號：2454 | 上市
供應鏈：NTN 通訊晶片（M90 數據機）
客戶：SpaceX Starlink（NTN 協議合作）+ 全球手機廠
```

| 指標 | 數值 |
|:---|:---:|
| M90 數據機 | 支援 NR-NTN 標準 |
| 全球 NTN 用戶 | 440 萬（美/加/日） |
| 6G NTN 定標 | 參與 3GPP Release 19 |

**重點摘要**：聯發科是 LEO 通訊的「鏟子銷售商」——不直接參與衛星製造或地面站，但提供 LEO 通訊關鍵晶片。Starlink Mobile 日增 ~18,000 用戶代表每台手機都有聯發科或競爭對手的 NTN 晶片機會。長期看 6G NTN 標準化將進一步擴大市場。

### 11. 全訊（5222）— 軍工級 GaAs 高功率 PA

