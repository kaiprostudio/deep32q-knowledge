## 二、技術路線分析（2026年5月更新）

### 2.1 頻段比較（Ku/Ka/V/E/W-band）

| 頻段 | 優點 | 缺點 | LEO星座採用現狀 |
|------|------|------|---------------|
| **Ku-band (12-18GHz)** | 技術成熟、天線尺寸適中 | 頻譜擁擠、容量有限 | Starlink第一代、OneWeb主要使用 |
| **Ka-band (26.5-40GHz)** | 高頻寬、天線尺寸小 | 雨衰較大、技術門檻較高 | Starlink Gen2/V2 Mini、Kuiper主要使用 |
| **V-band (40-75GHz)** | 超大頻寬、低干擾 | 技術門檻最高、雨衰嚴重 | 穩懋開發中、未來星座規劃 |
| **E-band (60-90GHz)** | 極大頻寬、低干擾 | 技術不成熟、距離短 | 地面回傳為主、衛星端開發中 |
| **W-band (75-110GHz)** | 極大頻寬 | 技術最不成熟 | 穩懋長期研發方向 |

### 2.2 雷射星間鏈路（ISL）技術成熟度

| 星座 | ISL採用現狀 | 技術細節 |
|------|-----------|---------|
| **Starlink** | ✅ V2 Mini已部署、Gen3 V3 H2 2026 Starship首發 | 雷射星間鏈路、跨軌道通訊、V3每發射60Tbps |
| **Kuiper/Leo** | ✅ 已部署（300+顆）| 雷射ISL、全球覆蓋、Kirkland廠日產4~5顆 |
| **OneWeb** | ❌ 第一代無、Gen2（340顆）規劃中（Airbus 2026/1簽約）| 未來可能納入 |
| **千帆/GW** | ✅ 部分部署（300+顆已發射）| 追趕中、中國140+發射目標（2026） |

### 2.3 Direct-to-Cell（直連手機）技術路線比較（2026更新）

| 方案 | Starlink Mobile | AST SpaceMobile | Lynk Global |
|------|----------------|-----------------|-------------|
| **技術基礎** | 10,296顆衛星、650+ D2C能力衛星、T-Mobile合作 | BlueBird衛星（New Glenn發射）、Verizon/AT&T合作 | 6顆在軌、50+ MNO全球協議、合併Omnispace |
| **目前服務** | SMS（美/新）、語音Beta、位置分享、日增~18,000用戶 | 測試階段（New Glenn發射失敗後重啟）、FCC商業許可已獲 | 6顆在軌、菲律賓測試中 |
| **2026目標** | 25M用戶（日增~52,000）、SMS+語音商用 | 45~60顆發射、美商用（New Glenn載荷已獲FCC批准）| 合併Omnispace、2027中商用 |
| **V2/V3能力** | 150Mbps（V2 Mini）、Gen3 V3每發射60Tbps、2026 H2 Starship | 120Mbps/cell（ASIC升級版BlueBird）| 多軌道架構（MEO+GEO回傳） |
| **商業化時程** | 2025下半年SMS、2028全面商用 | 2026年底美商用（45~60顆目標）、2027~28獲利 | 2027中商用、2028擴展 |
| **關鍵優勢** | 規模最大（10,296顆）、用戶基礎最廣、日增~18K | FCC商業許可、New Glenn載荷批准、$1.2B合約承諾 | 50+ MNO全球協議、多軌道架構 |
| **關鍵風險** | 頻譜收購$17B成本高、V3時程不確定（Starship延遲）| 發射失敗風險（New Glenn已失敗一次）、2027~28前無利 | 合併審批延遲、發射容量瓶頸 |

### 2.4 相控陣列天線技術與台廠能力

- **台灣優勢**：昇達科（天線/濾波器/OISL光星間鏈路）、穩懋（GaAs/GaN射頻晶片）在全球具備技術領先
- **關鍵技術**：低雜訊放大器（LNA）、濾波器/雙工器、相控陣列天線
- **台揚科技**：整合射頻+微波+自動雙軸追蹤技術的完整用戶終端設備開發中
- **雷射ISL**：昇達科OISL（Optical Inter-Satellite Link）組件已量產、光學背景台廠切入

### 2.5 6G NTN（Non-Terrestrial Network）標準進展

- **3GPP Release 17/18/19**：LEO標準化持續推進
- **聯發科M90數據機**：支援NR-NTN標準、美/加/日440萬用戶
- **SpaceX Starlink合作**：NTN協議開發、太空級AI運算晶片
- **2030年展望**：6G架構中衛星的角色將更加核心

## 四、投資總結（2026年5月更新）

### 4.1 核心觀點
1. **LEO產業已進入高速成長期**：2026年全球發射量超過4,800顆、Starlink用戶突破10M
2. **SpaceX IPO（2026/6）將引爆太空產值二次爆發**：預計籌資~750億美元、重塑市場評價基準
3. **Direct-to-Cell開啟消費級市場**：Starlink Mobile日增~18,000用戶、AST SpaceMobile獲FCC商業許可
4. **台灣供應鏈深度嵌入全球LEO生態**：從晶片→模組→天線→終端→材料，全鏈路參與
5. **「非紅供應鏈」優勢**：台廠成功嵌入SpaceX/Kuiper核心供應體系，避開地緣政治風險
6. **衛星汰換週期（2027-28）將放大需求**：2022年大量發射的衛星開始汰換、疊加新增部署量
