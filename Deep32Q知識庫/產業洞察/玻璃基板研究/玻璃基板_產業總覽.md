## 一、GCS（Glass Core Substrate）與 TGV（Through Glass Via）技術原理

### 1.1 什麼是玻璃基板？

玻璃基板（Glass Core Substrate / GCS）是以無鹼玻璃（Alkali-Free Glass）為基材的先進封裝載板。與傳統有機基板（如 ABF、BT 樹脂）不同，玻璃基板利用玻璃本身的物理特性——低熱膨脹係數（CTE）、超高平坦度、低介電常數（Dk）與低損耗角正切（Df）——來解決 AI/HPC 晶片日益嚴苛的封裝需求。

### 1.2 TGV（Through Glass Via）製程步驟

TGV 是玻璃基板的關鍵技術，透過玻璃基板建立垂直導電通道。製程分為以下步驟：

| 步驟 | 說明 | 關鍵技術 |
|:---:|------|----------|
| **1. 鑽孔（Via Formation）** | 在玻璃上鑽出微孔 | 超快雷射（飛秒雷射）、LIDE（雷射誘導深蝕刻）、乾式蝕刻（DRIE） |
| **2. 清潔與表面處理** | 去除碎屑、準備金屬化 | 濕式清洗、等離子清潔 |
| **3. 種子層沉積** | 在孔壁沉積導電種子層（銅） | PVD 濺鍍，需翻面確保孔壁完整覆蓋 |
| **4. 金屬化填充** | 電鍍銅填充孔洞 | 電解電鍍，要求無空洞、低電阻 |
| **5. 平坦化** | CMP 拋光使表面平整 | 化學機械拋光，暴露導電核心 |
| **6. RDL 圖案化** |  redistribution layer 光刻蝕刻 | 半導體級光刻，支援 sub-2µm 線寬 |

### 1.3 玻璃基板的六大優勢

| 特性 | 玻璃基板 | 傳統有機基板（ABF/BT） |
|------|---------|---------------------|
| **平坦度** | 超高（nm 級），適合薄膜沉積 | 較低，需額外拋光 |
| **耐熱性** | 250-400°C 低翹曲 | 高溫下易變形、翹曲 |
| **CTE（熱膨脹係數）** | 3-7 ppm/°C（接近矽） | 15-17 ppm（與矽不匹配） |
| **訊號損耗** | 極低 Dk（~5.5）、Df（~0.005） | 較高 Dk、Df，高頻損耗大 |
| **線寬能力** | sub-2µm（1.5µm 以下） | ~2/2µm 為限 |
| **散熱** | 低（1-2 W/mK），需額外方案 | 較佳，但高功率仍受限 |

## 四、T-Glass（T-Type 玻纖布）供應短缺分析

### 4.1 現況

- **日東紡（Nittobo）壟斷**：佔全球 T-Glass 約 **90%** 市場份額，NER-glass 佔 60-70%。
- **2026 年供需缺口**：預估 T-Glass 供應缺口達 **10%+**，比 2025 年更嚴重。
- **價格上漲**：日東紡 2025 年 8 月漲價 20%，預計 2026 年 4 月再漲 20-30%。
- **新產能時程**：大量新增產能最早 2027 年中才能上線。

### 4.2 對載板產業的影響

- **ABF 載板漲價**：Goldman Sachs 預估從 2025 Q4 到 2026 Q4，非 NVIDIA 的 ABF 載板價格將上漲 5-10%。
- **BT 載板受衝擊**：BT 基板用 T-Glass 可能面臨「雙位數百分比」的供應短缺。
- **優先權問題**：AI 基礎設施客戶（NVIDIA、Microsoft、Google、Amazon）獲得優先配額，消費電子、手機、汽車電子受到排擠。

### 4.3 替代方案

| 供應商 | 狀態 |
|--------|------|
| **台玻（1802）** | 低 DK、低 CTE 電子級玻纖布已切入 NVIDIA GB200 供應鏈，2026 年高階產線擴產 40-50% |
| **南亞塑膠** | 與日東紡合作 T-Type 玻纖布，目標 2027 年達到日東紡產能的 20% |
| **菲利華（PhiChem）** | 2025 年推出 M9 級石英電子布，已通過 NVIDIA 認證 |
| **中國廠商** | 宏和科技、河南廣源等已量產低介電布，但高階 T-Glass 仍待突破 |

## 五、FOPLP 與玻璃基板的關聯

### 5.1 FOPLP（Fan-Out Panel Level Packaging）

FOPLP 利用面板級封裝技術，以矩形玻璃面板取代傳統圓形矽晶圓，實現更高的整合密度和更低的單顆成本。

| 項目 | FOPLP + 玻璃基板 |
|------|-----------------|
| **面板尺寸** | G3.5 級（620mm × 750mm），約為 300mm 晶圓的 7 倍 |
| **全球 FOPLP 市場** | $3.8B（2025）→ $11.2B（2034） |
| **台灣角色** | 日月光投控、景碩、群創、友達積極布局；經濟部推動 FOPLP 生態系 |
| **TSMC** | CoPoS（Chip-on-Panel-on-Substrate）技術，2026 年建迷你產線 |
| **群創（Innolux）** | 全球最大 G3.5 FOPLP 面板，2025 年開始量產出貨 |
| **友達** | 利用面板製造經驗切入 FOPLP，與玻璃基板技術融合 |

### 5.2 中國面板廠轉型

- **京東方（BOE）**：8 吋試產線 → 全尺寸面板生產，2026 年後量產
- **Visionox**：投入 50 億人民幣，2025 下半年建立材料與設備供應鏈

## 六、主要風險因素

| 風險             | 說明                                        | 嚴重度    |
| -------------- | ----------------------------------------- | ------ |
| **量產延遲**       | 多數廠商量產時程從 2027-2028 延至 2030（如 LG Innotek） | ⚠️⚠️⚠️ |
| **良率瓶頸**       | 目前良率約 75-85%，成熟有機基板超過 95%；成本高出 30-50%     | ⚠️⚠️⚠️ |
| **玻璃脆性**       | 薄化至 100µm 以下極易破裂，製造線需特殊設備處理碎玻璃            | ⚠️⚠️   |
| **散熱問題**       | 玻璃導熱率僅 1-2 W/mK，高功率 (>50W/封裝) 散熱困難        | ⚠️⚠️   |
| **中國競爭**       | 京東方、Visionox 以國家補貼加速商業化，可能壓低價格            | ⚠️⚠️   |
| **ABF 反撲**     | ABF-GX 等新一代有機基板持續改進，可能延緩玻璃替代時程            | ⚠️     |
| **T-Glass 短缺** | 2026 年供需缺口 10%+，影響所有載板產能                  | ⚠️⚠️   |
