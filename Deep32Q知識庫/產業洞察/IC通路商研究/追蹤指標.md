## 六、追蹤日誌格式範例

每次更新追蹤日誌時，建議使用以下格式：

```
## 追蹤日誌 YYYY-MM-DD

### 營運面更新
- 文曄 X 月營收：XX 億（±X% YoY）
- 大聯大 X 月營收：XX 億（±X% YoY）
- ...

### 產業面更新
- 全球半導體市場最新預估：...
- 記憶體價格最新走勢：...

### 風險面更新
- 文曄負債比最新：XX%
- 匯率（USD/TWD）：XX

### 重大事件
- [日期] 事件摘要...
```

1. **各公司月營收公告** (每月): 官方發布之營運數據

4. **CSP 資本支出公告** (每季): 微軟、Google、AWS、Meta AI 投資

6. **Arrow / Avnet 財報** (每季): 歐美市場景氣指標
