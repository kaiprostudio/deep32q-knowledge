## 1. MLCC 技術原理與結構

MLCC（Multilayer Ceramic Capacitor，積層陶瓷電容）是最廣泛使用的被動元件。

### 基本結構
```
┌─────────────────────┐
│  外部電極（端電極）   │ ← 焊接至 PCB
├─────────────────────┤
│ 陶瓷介電層 (BaTiO₃) │ ← 儲存電荷
│ 內部電極（金屬層）   │ ← Ni/Cu/AgPd
├─────────────────────│ 重複疊層 100-3,000 層
│ 陶瓷介電層          │
├─────────────────────┤
│ 內部/外部電極       │ ← 焊接至 PCB
└─────────────────────┘
```

### 關鍵參數
- **容值**：pF → μF 級別
- **電壓等級**：6.3V、16V、50V、100V、200V、>1kV（HVDC）
- **尺寸**：1005 (402)、0603、0402、0201、01005
- **介電材質**：X7R（通用）、X5R、C0G/NP0（高頻低損耗）

## 3. 技術趨勢

### （1）小型化
- **尺寸演進**：1005 (4.0×2.0mm) → 0603 → 0402 → 0201 → **01005** (0.4×0.2mm)
- 01005 尺寸已量產，用於手機、可穿戴裝置

### （2）高容值
- 單顆 MLCC 容值突破 **100μF**（原為 μF 級）
- AI 伺服器需要大容量、低 ESR 的 MLCC

### （3）高電壓（HVDC 架構）
- **>1kV MLCC**：800V HVDC 架構推升需求
- AI 伺服器電源架構從 48V 轉向 400V+，高壓 MLCC 用量大增
- 禾伸堂已供應 NVIDIA Blackwell 平台高壓 MLCC

### （4）高耐溫
- AI 伺服器內部溫度可達 85°C+，需要高耐溫 MLCC
- X7R、X5S 材質持續優化

### （5）銅電極技術（華新科主導）
- 傳統：AgPd 內電極 → 成本高
- **銅電極**：華新科領先，成本降低 30-50%
- 銅電極 MLCC 已量產，2026 年持續擴充
