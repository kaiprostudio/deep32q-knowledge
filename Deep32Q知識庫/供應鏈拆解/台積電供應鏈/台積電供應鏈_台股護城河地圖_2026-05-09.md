# 台積電供應鏈·台股護城河地圖

> 深度研究報告 — 2026年5月
> 免責聲明：本報告僅供研究參考，不構成任何投資建議。

---

## 一、前言

台積電（2330）2025年全年營收達1,224.2億美元（年增35.9%），EPS 10.65美元（年增51.3%），2026年資本支出預計落在520至560億美元之間，未來三至四年累計上看2,000億美元。這一數字不僅刷新半導體產業歷史紀錄，更意味著AI算力基礎建設正式進入「鐵路基建」長週期。

台積電的每一美元資本支出，約有0.3至0.4美元流入台灣本地設備、材料、廠務與封測供應鏈。隨著先進製程（N2/A16）與先進封裝（CoWoS/SoIC）產能倍數擴張，台灣供應鏈廠商正從「配角」升級為「關鍵節點」——它們不僅是台積電的供應商，更是全球AI晶片製造生態中不可複製的護城河本身。

本報告聚焦18家具備結構性競爭優勢的台積電供應鏈公司，逐家拆解其護城河來源、財務體質與成長動能，並以護城河評級表進行橫向比較。

## 五、投資策略總結

### 5.1 三大投資主軸的特性分析

| 類別 | 設備股 | 耗材/材料股 | IP/設計股 |
|:---|:---|:---|:---|
| **營收能見度** | 中（依設備交機時程） | 高（持續消耗性需求） | 高（權利金長期穩定） |
| **護城河本質** | 技術門檻+認證週期 | 認證壁壘+不可替代性 | 專利授權+生態系統鎖定 |
| **週期性** | 中等（與資本支出連動） | 低（半導體產出量連動） | 極低（IP授權長尾效應） |
| **毛利率** | 25-40% | 35-50% | 80-100% |
| **成長爆發力** | 高（擴產週期同步） | 中高（穩定成長） | 中（累積性成長） |
| **下行風險** | 擴產遞延 | 技術迭代 | 新技術替代 |

### 5.2 各類投資風格適合的標的

**積極成長型**（高風險高報酬）：
- 新應材(4749) — 2nm量產+KrF光阻2026年雙引擎
- 弘塑(3131) / 辛耘(3583) — CoWoS擴產直接受惠
- 均華(6640) — 營收爆發力強，但基期較低波動大

**穩健護城河型**（低波動長期持有）：
- 家登(3680) — 全球獨佔+結構性成長
- 中砂(1560) — 2nm獨家供應商+耗材剛性需求
- 京鼎(3413) — 評價合理+配息穩定(現金殖利率約4.8%)

**高毛利輕資產型**（毛利率>80%）：
- 力旺(3529) — 100%毛利率+專利長尾
- M31(6643) — IP授權業務毛利高

**產業龍頭型**（市值大流動性佳）：
- 聯發科(2454) — 手機+AI ASIC雙支柱
- 日月光投控(3711) — 封測全球龍頭
- 環球晶(6488) — 矽晶圓全球前三

### 5.3 核心觀察框架

```
台積電資本支出 → 設備股拉貨 (前6-12月)
       ↓
CoWoS產能開出 → 耗材/材料股 (同步放量)
       ↓
晶片量產 → IP權利金收入 (長期穩定)
       ↓
封測委外 → 封測股 (落後6-12月)
```

**關鍵指標**：
1. 台積電法說會資本支出預估 — 決定設備股訂單強度
2. 台積電CoWoS月產能目標 — 決定封裝設備/耗材需求
3. 台積電N2/A16良率爬坡進度 — 決定新應材、中砂等先進材料出貨
4. 全球AI晶片出貨量 — 決定穎崴、旺矽測試介面需求

### 5.4 風險提示

- **技術迭代風險**：新材料/新技術可能取代現有供應鏈地位
- **地緣政治風險**：美中科技管制、關稅政策影響全球供應鏈配置
- **匯率風險**：新台幣升值將侵蝕以外幣計價的營收與獲利（家登、中砂等影響明顯）
- **擴產遞延風險**：台積電建廠進度若延遲，將直接影響設備/材料出貨時程
- **客戶集中風險**：多數公司高度依賴台積電單一客戶
- **評價偏高风险**：部分供應鏈公司本益比偏高（如穎崴64倍、力旺>60倍），需留意修正壓力
