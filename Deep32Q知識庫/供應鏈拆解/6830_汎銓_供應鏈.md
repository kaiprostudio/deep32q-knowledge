# 6830 汎銓科技 — 供應鏈拆解

> 資料來源：各股摘要（2026-04-26）| 審計報告 F01-F18 + G01-G14

---

## 一、公司定位與產業角色

汎銓科技（6830）為台灣領先的**半導體高階材料分析 (MA) 與故障分析 (FA)**服務商，成立於2005年。

**產業定位**：半導體製程後段「檢測與分析」利基市場，不直接生產晶圓或封裝，而是提供**第三方檢測服務 + 自主設備銷售 + 技術授權**。

**全球佈局**：台灣、中國（南京/深圳/上海）、日本川崎、美國矽谷共12個營運據點，貼近客戶研發中心。

## 二、上游供應商拆解

### 2.1 關鍵分析設備（核心資產）

| 設備類型 | 供應商 | 說明 |
|:---|:---|:---|
| **SEM（掃描式電子顯微鏡）** | Thermo Fisher Scientific、Hitachi High-Tech | 材料分析核心設備，用於微觀結構觀察 |
| **TEM（穿透式電子顯微鏡）** | Thermo Fisher Scientific、JEOL | 高解析度晶格結構分析，用於先進製程（2奈米以下） |
| **FIB（聚焦離子束）** | Thermo Fisher Scientific、Hitachi High-Tech | 截面製備與微觀成像，FA關鍵設備 |
| **XRF（X射線螢光光譜儀）** | Malvern Panalytical、Rigaku | 元素成分分析，化學組成檢測 |
| **SIMS（次離子質譜儀）** | Thermo Fisher Scientific、Cameca | 微量雜質與摻分佈分析，半導體材料純度檢測 |
| **AES/XPS（電子能譜/X射線光電能譜）** | Kratos、Thermo Fisher Scientific | 表面化學分析，薄膜厚度與成分鑑定 |
| **光學量測設備** | KLA、ASML（Yokogawa） | 光学缺陷檢測、膜厚量測 |
| **矽光子測試平台（MSS HG）** | **自主研發** | 每套售價4,000-6,000萬元，整合光學量測、影像判讀與精密機構控制 |

> **關鍵洞察**：汎銓大量資本支出（過去三年半40億）主要用於購置上述高階分析設備。設備供應商高度集中於**Thermo Fisher Scientific（市佔最大）**與**Hitachi High-Tech**。

### 2.2 化學品與耗材

| 類別 | 主要供應商（推估） | 用途 |
|:---|:---|:---|
| **高純度氣體**（Ar、N₂、H₂等） | 台灣氣體工業、林德氣體(Linde)、Air Liquide | SEM/TEM/FIB離子源、純化環境 |
| **電子級化學品**（溶劑、蝕刻液）| 日本信越化學、台灣南亞科技、台積電供應體系 | 樣品前處理、截面製備 |
| **標靶材料**（SIMS分析用） | 美國SPI Supplies、日本Kuramoto | 質譜儀校準與標準品 |
| **電子顯微鏡樣品台/載具** | 各國設備原廠（Thermo、JEOL等）| TEM/SEM專用樣品載具 |
| **濾膜與過濾材料** | 日本Milmix、美國Poretics | 薄膜樣品製備 |
| **真空系統耗材**（泵油、密封圈）| Edwards Vacuum、Pfeiffer Vacuum | 維持設備真空環境 |

### 2.3 零組件與系統整合

| 類別 | 說明 |
|:---|:---|
| **精密機構零組件** | MSS HG設備所需的精密機械結構、光學鏡頭、感測器 |
| **電子元件** | 訊號處理板卡、高頻電路模組（矽光子測試用）|
| **軟體授權** | 影像分析軟體、數據處理平台（部分為自主開發）|
| **廠房設施** | 台元科技園區新廠、南京/深圳/上海據點（2026年8月完工）|

## 三、下游客戶拆解

### 3.1 核心客戶群（營收結構）

| 客戶類型 | 代表企業 | 佔比預估 | 說明 |
|:---|:---|:---:|:---|
| **半導體製造廠（Foundry）** | **台積電（TSMC）為核心** | ~40-50% | 先進製程（2奈米以下）材料分析需求最大，台積電為最大單一客戶 |
| **記憶體廠** | 美光（Micron）、三星、華邦電 | ~15-20% | 3D NAND、HBM封裝檢測需求 |
| **IC設計公司** | 輝達（NVIDIA）、高通、聯發科 | ~15-20% | 新產品開發階段FA需求，美國Tier-1 AI巨頭已在竹北廠設專屬專區 |
| **封測廠** | 日月光、矽品、安靠（Amkor） | ~10-15% | 封裝可靠度驗證、失效分析 |
| **設備商** | KLA、ASML、Applied Materials | ~5-10% | 新設備開發驗證、客戶支援分析服務 |
| **矽光子/光通訊公司** | Intel、Broadcom、Ayar Labs、台灣矽子科技 | ~5-10% | 光進光出分析（市佔>90%），CPO檢測需求快速成長 |

### 3.2 客戶結構特點

- **分散但高質量**：雖台積電為核心客戶，但整體客戶結構相對分散
- **Tier 1領導廠商為主**：目標客群鎖定半導體產業龍頭
- **美國AI巨頭深度合作**：已在汎銓竹北廠設立專屬AI客戶專區，參與未來2-3年新產品早期開發
- **全球佈貼近研發中心**：12個據點分佈於主要半導體集群區

## 四、關鍵設備與技術壁壘

### 4.1 核心技術能力矩陣

| 技術領域 | 市場地位 | 關鍵設備/方法 |
|:---|:---|:---|
| **高階材料分析（MA）** | 國內市佔>50% | SEM/TEM/FIB/XRF/SIMS/AES/XPS全套分析平台 |
| **矽光子「光進光出」分析** | 全球市佔>90%（絕對領先）| 自主光損偵測技術（台灣發明專利I870008B）|
| **MSS HG矽光子測試平台** | 商業化初期（第3套組裝完成）| 整合光學量測、影像判讀與精密機構控制 |
| **故障分析（FA）** | 國內主要供應商之一 | FIB截面製備、EDS元素分析、熱顯微技術 |
| **可靠度驗證（RA）** | 次要業務 | 高溫高濕、熱循環、HAST等標準測試 |
| **CPO檢測設備授權** | 新業務（5%營收佔比）| 光電整合封裝檢測技術授權 |

### 4.2 技術壁壘分析

1. **設備投資門檻極高**：單台TEM/SEM可達數千萬至億級台幣，過去三年半40億Capex
2. **技術人才稀缺**：創辦人柳紀綸+技術長陳榮欽博士+營運長廖永順「經營鐵三角」合作逾20年
3. **專利保護**：光損偵測裝置台灣發明專利I870008B，台、日、美專利佈局
4. **客戶信任深度**：與台積電等Tier 1長期合作，參與新產品早期開發階段

## 五、成本結構分析

### 5.1 營收構成與成本分攤（2026年預估）

| 業務板塊 | 營收佔比 | 成本結構特徵 |
|:---|:---:|:---|
| **半導體材料分析(MA)服務** | ~50% | 人力成本（技術工程師）+ 設備折舊 + 化學品耗材 |
| **矽光子分析服務** | ~20% | 高階設備折舊佔比高，但邊際成本隨規模下降 |
| **MSS HG檢測設備銷售** | ~15% | 硬體BOM成本 + 研發分攤 + 組裝測試成本（每套4,000-6,000萬）|
| **故障分析(FA)與可靠度驗證** | ~10% | 人力密集，設備共用程度高 |
| **CPO檢測設備授權** | ~5% | 極低邊際成本（純技術授權權利金）|

### 5.2 主要成本項目

| 成本類別 | 佔比預估 | 說明 |
|:---|:---:|:---|
| **折舊費用** | ~30-35% | 過去三年半40億Capex，年折舊約7.77億（2025年），佔CFO 104.5% |
| **人力成本** | ~25-30% | 高階技術工程師團隊，全球12據點人力投入 |
| **設備採購/維護** | ~15-20% | 高階分析設備購置、保養合約、零組件更換 |
| **化學品與耗材** | ~8-12% | 高純度氣體、電子級化學品、標靶材料 |
| **廠房與營運** | ~5-8% | 台元科技園區、海外據點租金/折舊、水電 |
| **研發投入** | ~5-8% | MSS HG設備開發、矽光子技術迭代、CPO新業務 |

### 5.3 毛利率趨勢與驅動因素

| 年度 | 毛利率 | 關鍵驅動因素 |
|:---|:---:|:---|
| 2022 | **44.6%** | 純服務模式，高附加價值 |
| 2023 | ~36.8% | 設備投資增加，折舊上升 |
| 2024 | ~23-26% | 全球佈局成本高峰，規模擴張但定價權不足 |
| 2025 | **28.6%** | Q4回升至28.6%（較2024Q4的22.7%改善），成本控制初見成效 |

> **⚠️ 關鍵問題**：2022-2025年營收成長+25.4%，但毛利率從44.6%降至28.6%（-16.0ppt），呈現「量價背離」。

## 六、議價能力分析

### 6.1 對上游供應商的議價能力：**中偏低**

| 因素 | 影響方向 | 說明 |
|:---|:---:|:---|
| **設備採購規模** | ⬆️ 正面 | 過去三年半40億Capex，對Thermo Fisher等大廠有一定議價空間 |
| **設備供應商集中度高** | ⬇️ 負面 | SEM/TEM/FIB市場高度集中於Thermo Fisher、Hitachi、JEOL，替代選擇有限 |
| **化學品/耗材** | ⬇️ 負面 | 高純度氣體、電子級化學品有台灣本土供應商可替代，但量不大時議價力弱 |
| **自主設備（MSS HG）** | ⬆️ 正面 | MSSHG為自主研發，降低對外部設備商的依賴，長期有利 |

### 6.2 對下游客戶的議價能力：**中**（正在提升）

| 因素 | 影響方向 | 說明 |
|:---|:---:|:---|
| **矽光子90%市佔率** | ⬆️ 正面 | 絕對領導地位帶來強大議價權，客戶轉換成本高 |
| **台積電核心客戶依賴** | ⬇️ 負面 | 單一最大客戶訂單波動影響顯著，需維持高服務品質 |
| **技術壁壘與專利** | ⬆️ 正面 | 光損偵測專利、全套分析平台形成進入壁壘 |
| **服務+設備+授權模式** | ⬆️ 正面 | 從純服務轉型為三軌並進，設備銷售提升定價權 |
| **量價背離現狀** | ⬇️ 負面 | 2024年毛利率低點僅22.7%，反映定價能力仍待驗證 |

### 6.3 議價能力綜合評估

| 維度 | 評級 | 趨勢 |
|:---|:---:|:---:|
| 對設備供應商 | C+ | ⬆️ 隨MSSHG量產逐步改善 |
| 對化學品/耗材供應商 | C | ➡️ 穩定但提升空間有限 |
| 對半導體製造廠客戶 | B+ | ⬆️ 矽光子領導地位強化 |
| 對IC設計/AI客戶 | A- | ⬆️ 美國AI巨頭深度合作，早期參與提升黏著度 |
| **綜合議價能力** | **B** | ⬆️ 從「純服務」到「三軌並進」轉型中，預期逐步提升 |

## 九、關鍵結論

1. **上游依賴度高**：核心分析設備集中於Thermo Fisher等少數大廠，但40億Capex已建立完整分析平台
2. **矽光子絕對領導**：全球市佔>90%的「光進光出」分析能力，是議價權最強的護城河
3. **客戶結構健康**：以Tier 1半導體龍頭為主，美國AI巨頭深度合作提升黏著度
4. **量價背離待解**：毛利率從44.6%降至28.6%，需透過設備銷售（15-23%佔比）與授權模式改善
5. **轉型關鍵期**：從「純服務」到「服務+設備+授權」三軌並進，2026年為矽光子量產元年與MSSHG設備出貨關鍵年份
