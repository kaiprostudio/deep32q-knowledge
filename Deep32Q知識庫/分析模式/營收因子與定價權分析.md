# F01 營收因子與定價權分析模式（連接器產業專用版）

> **適用**：連接器 / 電源線組 / 電子零組件製造業
> **首次應用**：6290 良維（2026-04-24）
> **升級日期**：2026-04-27（整合 C01 連接器產業框架精華）
> **核心命題**：連接器公司的營收不在「做多少產品」，而在於它在供應鏈中的「不可替代性位置」。

---

## 一、營收結構轉型評估（通用框架）

### 1.1 傳統 vs 新興業務拆解

| 維度 | 傳統業務 | 新興業務 |
|:---|:---|:---|
| **典型產品** | 消費電子電源線 / 一般連接器 | 數據中心 / AI伺服器電源連接器 |
| **毛利特徵** | 低毛利（15-20%）、週期性強 | 高毛利（30-42%）、結構性需求 |
| **客戶週期** | 消費電子換機週期（18-24個月） | AI Capex 投資週期（3-5年） |
| **定價權** | 弱 — 同業競爭激烈、可替代性高 | 強 — 認證壁壘 + 設計進入點 |

### 1.2 營收結構轉型評估指標

- **各產品線營收佔比變化**：傳統 vs 新興業務的結構性位移
- **YoY 成長差異**：各產品線年增率差距（反映轉型速度）
- **營收貢獻結構圖**：Σ(各產品線佔比 × 該產品線毛利率)

## 二、定價權量化指標（通用框架）

| 指標 | 評估方法 | 連接器產業標準 |
|:---|:---|:---|
| **毛利率趨勢** | 從低點到高點的改善幅度（Δ%） | 良維：22.8%→34.8%（+12pp） |
| **季度毛利率走勢** | 連續改善的穩定性 | Q4 創高確認結構性改善 |
| **成本轉嫁能力** | 銅價上漲期間毛利率是否維持或提升 | 良維：有轉嫁機制但存在時差 |
| **規模經濟** | 營收規模擴大對固定成本攤提的影響 | 良維跨越80億臨界點後，每增20億挹注~4.9億營業利益 |

## 三、營收驅動因子權重分配（通用框架）

### 3.1 驅動因子拆解方法

將營收來源拆解為多個驅動因子，估算各因子佔比與成長動能：

```
營收 = Σ(各驅動因子營收)
     = Σ(各因子客戶數 × 單客貢獻 × 成長率)
```

### 3.2 結構性 vs 週期性評估矩陣

| 因子類型 | 特徵 | 估值倍數 | 風險等級 |
|:---|:---|:---:|:---:|
| **結構性** | 長期趨勢明確、可預測3-5年 | 較高（20x+ PE） | 低 |
| **週期性** | 景氣循環驅動、波動大 | 較低（10-15x PE） | 高 |
| **混合型** | 部分結構性 + 部分週期性 | 取中 | 中 |

## 四、風險因子矩陣（通用框架）

| 維度 | 評估項目 |
|:---|:---|
| **客戶集中度** | 單一/前幾大客戶佔比（良維~25% Apple、瀚荃~60-70% EMS大廠） |
| **原物料波動** | 銅價等大宗原料轉嫁機制與時差 |
| **產業週期** | AI Capex、消費電子景氣循環的疊加效應 |
| **匯率風險** | 美元計價營收 vs 台幣報稅（生產基地分散度影響） |
| **規格迭代** | GB200→GB300→Vera Rubin 的連接器規格變更風險 |

## 五、連接器產業專用：營收驅動因子模型（Revenue Driver Model）

### 5.1 連接器營收六驅力模型

```
營收 = f(客戶數, 單客貢獻, 產品組合, 產能利用率, 規格迭代乘數)
```

| 驅動因子 | 評估問題 | 量化指標 | 良維實例 | 瀚荃實例 |
|:---|:---|:---|:---:|:---:|
| **① 客戶數** | 直接/間接供應的終端客戶總數？CSP數量與擴產節奏？ | 前十大客戶佔比、新增客戶數 | ~25% Apple + CSP多頭並進 | 透過EMS間接、前十大~60-70% |
| **② 單客貢獻** | 單一客戶年均採購金額？是否有成長趨勢？ | 單客營收（億元）、YoY | Apple + CSP雙引擎 | AI訂單看到2026年7月 |
| **③ 產品組合** | AI/數據中心產品佔比多少？高毛利產品占比趨勢？ | 各應用領域營收佔比 × 該領域毛利率 | ~50% AI/DC、~25% Apple、~15% EV | AI朝30%但筆電仍23% |
| **④ 產能利用率** | 各產品線利用率？接近甜蜜點（70-80%）？ | 實際產量 / 設計產能 | 泰國廠70-80%（甜蜜點） | AI滿載但消費性產線65-75% |
| **⑤ 規格迭代乘數** | GB200→GB300的連接器用量倍數？每機櫃連接器數量變化？ | 每機櫃連接器總數、倍數成長 | GB200~143→GB300~515（3.6x） | 同上，板端寡佔受益更大 |
| **⑥ 認證壁壘** | 新客戶取得全套認證需耗時多久？轉換成本多高？ | 認證週期（月）、存量認證數量 | 50年認證存量、新進者12-24個月+ | 伺服器級認證、板端寡佔 |

### 5.2 營收驅動因子權重評分卡（0-30分）

| 評估項目 | 評分標準 (0-5) | 良維現況 | 瀚荃現況 |
|:---|:---|:---:|:---:|
| **客戶多元化** | 0=高度集中, 5=分散多領域 | 4（Apple+CSP多頭） | 3（EMS集中度高） |
| **單客成長動能** | 0=萎縮, 5=快速擴張 | 4（CSP Capex複合38%） | 4（AI訂單可見度高） |
| **產品組合升級** | 0=傳統為主, 5=AI佔比>40% | 5（~50% AI/DC） | 3.5（AI朝30%但筆電拖累） |
| **產能利用率** | 0=低利用率, 5=甜蜜點以上 | 4（70-80%） | 3.5（AI滿載但消費性偏低） |
| **規格迭代乘數** | 0=無成長, 5=3x以上倍數 | 4（GB300為GB200的3.6x） | 5（板端每機櫃用量倍增） |
| **供應鏈定位** | 0=外部/間接, 5=直接+板端 | 4（直接CSP+Apple） | 3.5（透過EMS但板端寡佔） |
| **總分** | — | **24/30** | **24/30** |

> ⚠️ 兩家總分相同但驅動結構不同：良維勝在「直接供應+高AI佔比」，瀚荃強在「板端寡佔+規格迭代乘數」。

## 六、連接器產業專用：毛利率驅動因子拆解（Gross Margin Drivers）

### 6.1 毛利率六驅力模型

```
毛利率 = f(產品組合, 規模經濟, 原材料成本, 定價權, 產能利用率, 匯率)
```

| 驅動因子 | 評估問題 | 量化指標 |
|:---|:---|:---|
| **① 產品組合** | AI/數據中心產品佔比多少？高毛利產品占比趨勢？ | 各應用領域營收佔比 × 該領域毛利率 |
| **② 規模經濟** | 是否跨越臨界營收？每增20億挹注多少營業利益？ | 營收規模 vs 臨界點（良維80億） |
| **③ 原材料成本** | 銅材佔COGS比重？是否有轉嫁機制？轉嫁時差多長？ | 銅價波動敏感度、轉嫁條款內容 |
| **④ 定價權** | 寡佔格局 vs 紅海競爭？客戶議價能力對比？ | 毛利率同業比較、報價週期 |
| **⑤ 產能利用率** | 各產品線利用率？接近甜蜜點（良維70-80%）？ | 產能利用率 × 固定成本分攤效果 |
| **⑥ 匯率影響** | 營收計價幣別 vs 成本幣別？生產基地分散度？ | 台幣/美金匯率敏感度分析 |

### 6.2 毛利率預測公式

```
預估毛利率 = Σ(各產品線佔比 × 該產品線毛利率) + 規模經濟加成 - 原材料波動 - 匯率影響
```

**實例：良維2025年毛利率31.4%的拆解**

| 產品線 | 佔比 | 預估毛利率 | 貢獻 |
|:---|:---:|:---:|:---:|
| 數據中心/AI（~50%） | 50% | ~38-42% | ~19.0-21.0% |
| Apple供應鏈（~25%） | 25% | ~30-34% | ~7.5-8.5% |
| 電動車充電（~15%） | 15% | ~28-32% | ~4.2-4.8% |
| 其他（~10%） | 10% | ~25-28% | ~2.5-2.8% |
| **加權平均** | **100%** | — | **~33.2-37.1%** |
| 減：固定成本分攤、轉嫁時差、匯率影響 | — | -1-3% | 實際31.4%（Q4達34.8%） |

> ⚠️ 瀚荃2025年毛利率37.41%（Q4達39.1%），高於良維，反映其寡佔格局帶來的定價權溢價。

### 6.3 毛利率趨勢判斷規則

| 信號 | 意義 | 行動建議 |
|:---|:---|:---|
| **毛利率逐季上升** | 產品組合升級成功 + 規模經濟顯現 | 正面，可給予較高估值 |
| **毛利率穩定但同業偏低** | 定價權不足或成本結構劣勢 | 需關注競爭動態 |
| **毛利率突然下滑** | 產品組合惡化 / 原材料飆升 / 產能利用率下降 | 立即檢查驅動因子 |
| **Q4毛利率創新高** | 旺季效應 + 產品組合優化雙重驅動 | 確認是否結構性改善 |
| **毛利率同業最高** | 護城河有效轉化為溢價能力 | 長期持有標的 |

## 七、連接器產業專用：營收驅動因子實例分析

### 7.1 良維（6290）營收驅動因子深度拆解

**營收結構：**
- 2023 低點：73.6 億（毛利率 22.8%）
- 2025 高峰：101.6 億（毛利率 31.4%）
- 2026 展望：營收突破 120 億，毛利率維持 30-35%

**六大驅動因子量化：**

| 驅動因子 | 2023（低點） | 2025（高峰） | 變化方向 | 影響程度 |
|:---|:---:|:---:|:---:|:---:|
| **① 客戶數** | CSP起步階段 | 多頭並進（Amazon+Google+Meta） | ↑ 正面 | 中等 |
| **② 單客貢獻** | 低（轉型初期） | Apple+CSP雙引擎放量 | ↑↑ 正面 | 大 |
| **③ 產品組合** | ~30% AI/DC | ~50% AI/DC | ↑↑ 正面 | 大 |
| **④ 產能利用率** | ~60-65%（低點） | 70-80%（甜蜜點） | ↑ 正面 | 中等 |
| **⑤ 規格迭代** | GB200初期 | GB300放量（3.6x乘數） | ↑↑ 正面 | 大 |
| **⑥ 認證壁壘** | 50年存量優勢 | 直接CSP+Apple雙通道 | → 穩定 | 基礎 |

**營收成長歸因：**
- 產品組合優化（30%→50% AI佔比）：貢獻約 +18 億
- GB300規格迭代乘數（3.6x）：貢獻約 +12 億
- 客戶多元化（CSP多頭並進）：貢獻約 +8 億
- 產能利用率提升（65%→75%）：貢獻約 +4 億
- **合計**：~73.6→101.6 億（+28 億，YoY +38%）

### 7.2 瀚荃（8103）營收驅動因子深度拆解

**營收結構：**
- AI 營收占比：~18-21% → 30%（FY2026E）
- 毛利率：37.41%（Q4達39.1%），高於良維
- 筆電業務仍佔 ~23%，為主要拖累因子

**六大驅動因子量化：**

| 驅動因子 | 現況 | 趨勢 | 影響程度 |
|:---|:---|:---|:---:|
| **① 客戶數** | 透過EMS間接供應（前十大~60-70%） | AI訂單可看到2026年7月 | 中等（集中度高但穩定） |
| **② 單客貢獻** | AI出貨放量中 | GB300放量帶動 | ↑ 正面 |
| **③ 產品組合** | AI朝30%但筆電仍23% | 結構性改善中 | ↑ 正面（但速度較良維慢） |
| **④ 產能利用率** | AI滿載但消費性產線65-75% | 結構性分化 | → 中性（AI與消費性兩極） |
| **⑤ 規格迭代** | 板端每機櫃用量倍增（3.6x） | GB300放量中 | ↑↑ 正面（板端寡佔受益最大） |
| **⑥ 認證壁壘** | 伺服器級認證、板端寡佔僅1家對手 | 護城河深厚 | → 穩定（高） |

**瀚荃 vs 良維驅動因子差異：**
- 良維：AI佔比更高（50% vs ~30%），直接供應優勢明顯
- 瀚荃：毛利率更高（37.41% vs 31.4%），板端寡佔帶來定價權溢價
- 瀚荃的「規格迭代乘數」評分最高（5/5），反映其在板端高功率連接器的不可替代性

## 九、營收驅動因子監控清單（Operating Checklist）

| 檢查項目 | 正面信號 | 負面信號 | 監控頻率 |
|:---|:---|:---|:---:|
| **CSP Capex** | Amazon/Google/Meta AI基建持續擴張 | 任一CSP砍Capex >10% | 每季 |
| **產品組合** | AI/數據中心佔比逐季提升 | 低毛利消費電子佔比回升 | 每季 |
| **產能利用率** | 各產品線接近或超過甜蜜點（70-80%） | 產能利用率持續低於65% | 每季 |
| **訂單能見度** | 3-4個月以上訂單能見度 | 訂單能見度縮短至1個月內 | 每月 |
| **毛利率趨勢** | 逐季上升或穩定在高檔 | 連續兩季下滑超過2個百分點 | 每季 |
| **GB300認證進度** | 通過認證並量產 | 認證延遲或規格變更 | 每季 |
| **規格迭代（Vera Rubin）** | 與現行兼容或升級 | 全新設計 → 重新認證風險 | 每季 |
| **寡佔格局穩定性** | 僅一家未上市櫃對手 | 同業取得認證或毛利率追平 | 每季 |
