# F06: 存貨與營運週期分析框架

> 跨公司通用的審計分析方法。適用於連接器、電子零組件製造業，可擴展至有存貨管理的製造業。
> 基於良維（6290）與瀚荃（8103）實證報告提煉。

---

## 📋 審計題目

1. **存貨金額變動趨勢為何？** — 水位是否合理？
2. **營運週期效率如何？** — 去化速度是否穩定或惡化？
3. **流動性資產是否足以覆蓋短期與長期債務？** — 償債防禦力？

---

## 🎯 專家直斷（通用模板）

> _「存貨從 [起始值] 升至 [最新值]（+[X]%），[週轉天數/率] 維持在 [Y]，去化速度 [穩定/放緩/加速]。現金 [Z] 元覆蓋有息負債 [W] 倍，流動比 [A]%、速動比 [B]%，償債防禦力 [充足/緊繃]。評級 **X（等級）**：存貨風險可控，惟 [關鍵警訊]。」_

---

## 📊 診斷評級標準

| 評級 | 存貨週轉天數 vs 同業 | 存貨/營收增速比 | 流動比率 | 現金/有息負債 | 綜合判定 |
|:---:|:---|:---|:---|:---|:---|
| **A（優）** | 等於或低於同業中位數 | < 1.0（存貨增速 < 營收增速） | > 250% | > 5x | 存貨健康，流動性充裕 |
| **A-（良）** | 同業中位數 ±15% | 1.0–1.5（存貨增速略快） | > 200% | > 3x | 存貨可控，需追蹤去化 |
| **B（警）** | 高於同業中位數 15–30% | 1.5–2.0（存貨增速明顯快） | 150–200% | 1–3x | 存貨壓力上升，需關注跌價 |
| **C（危）** | 高於同業中位數 > 30% | > 2.0（存貨增速遠超營收） | < 150% | < 1x | 流動性風險，潛在跌價衝擊 |

**評級調整因子：**
- 單季存貨 QoQ 變動 > ±20%：自動降半級（A→A-，A-→B）
- 連續兩季存貨增加且營收增速 < 5%：自動降一級
- 有息負債/總資產 > 30%：自動降半級

---

## 🔍 分析框架（四層診斷）

### 第一層：存貨金額變動趨勢

**核心指標：**
- 各季度存貨絕對值（千/百萬元）
- 佔總資產比、佔流動資產比
- 較上季變動百分比（QoQ）

**分析要點：**
1. **長期趨勢**：2–3 年存貨走勢（上升/下降/波動）
2. **結構性轉折**：單季 QoQ 異常變動（> ±15%）需標註原因
3. **佔比變化**：存貨/總資產 > 20% 或存貨/流動資產 > 35% 需特別關注
4. **與營收匹配度**：存貨增速 vs 營收增速的比率（> 1.5 為警訊）

**輸出格式：**

| 季度 | 存貨金額 | 較上季變動 | 佔總資產比 | 佔流動資產比 | 備註 |
|:---|:---|:---|:---|:---|:---|

### 第二層：存貨週轉效率

**核心指標：**
- 存貨週轉率 = 年度營收 / 平均存貨（期初+期末）/2
- 存貨週轉天數 = 365 / 存貨週轉率

**分析要點：**
1. **年度比較**：連續 2–3 年週轉天數是否穩定
2. **同業對照**：與產業同業中位數比較（連接器/電子零組件約 70–90 天）
3. **季節性調整**：旺季備貨可能暫時拉高 QoQ，但年度週轉天數應穩定
4. **惡化預警**：若營收增速趨緩而存貨維持高位，週轉天數將被動拉長

**輸出格式：**

| 年度 | 營收 | 平均存貨 | 存貨週轉率 | 週轉天數 | vs 同業中位數 |
|:---|:---|:---|:---:|:---:|:---|

### 第三層：流動性對債務的覆蓋能力

**核心指標：**
- 流動比率 = 流動資產 / 流動負債
- 速動比率 =（現金 + 應收帳款 + 短期投資）/ 流動負債
- 現金 / 有息負債 =（現金 + 約當現金）/（短借 + 長借 + 公司債）
- 速動資產 / 有息負債（更嚴格的流動性測試）

**分析要點：**
1. **短期覆蓋**：流動比率 > 200% 為安全線，> 250% 為充裕
2. **速動測試**：排除存貨後的流動性，> 150% 為安全
3. **有息負債覆蓋**：現金 / 總借款 > 5x 為充裕，> 3x 為安全
4. **槓桿水平**：負債比 < 30% 為保守，> 50% 需關注
5. **利息保障**：稅前淨利 / 利息費用 > 10x 為安全

**輸出格式：**

| 季度 | 流動比 | 速動比 | 現金(億) | 短借(億) | 長借(億) | 現金/總借款 |
|:---|:---|:---|:---|:---|:---|:---|

### 第四層：跌價損失風險評估

**核心指標：**
- 存貨週轉天數 vs 同業比較（超過同業 30% 為警訊）
- 存貨結構：製成品 / 在製品 / 原料佔比（需管理層提供）
- 跌價損失提列比率：存貨跌價損失 / 存貨總額

**分析要點：**
1. **景氣敏感度**：終端需求（消費性電子 / 伺服器 / 汽車）對存貨跌價的影響
2. **備貨合理性**：跳增是否為已知訂單備貨（法說會確認）或無明確需求的堆積
3. **歷史平滑度**：單季存貨變動超過 ±20% 且無明確原因，依 3σ 統計法則列為高優先級審計項目
4. **同業對照**：比較同業存貨提列比率和週轉天數

## 💡 專家評析模板（雙重視角）

### 華爾街空頭視角（Skepticism）

> _假設 [景氣條件] 下，[公司名] [X] 億存貨將面臨跌價風險。假設製成品佔比 [Y]%（約 [Z] 億），在需求萎縮 [W]% 情境下，潛在跌價損失達 [A-B] 億，可吃掉單季淨利 [C]%。但現金防禦牆 [D] 億足以吸收此衝擊，[是否觸發流動性危機]。_

**空頭視角延伸：**
- 產業存貨跌價準備提列常落後景氣轉折一至兩季
- 若營收增速放緩而公司未同步提列跌價損失，將出現「存貨帳面價值高估、淨利虛胖」問題
- 單季極端異常值（> ±20% QoQ）缺乏歷史平滑曲線支撐，應列為高優先級審計項目

### 物理對策與觀測節點

1. **追蹤 [X] 存貨是否回降至 [Y]**——持續超過 [Z] 即「備貨失靈」訊號
2. **監控應收帳款週轉天數是否同步拉長**——AR 與存貨同時積壓意味終端實質降溫
3. **[短借/長借] 從 [A] 增至 [B]**，需確認是否為季節性週轉；若持續攀升將壓縮利息保障空間
4. **要求管理層提供存貨明細**（製成品/在製品/原料佔比）及 [年度] 備貨預測，以判別跳增合理性

**觀測節點：**
- **[季度] 季報發布後**：重點檢視存貨絕對值與週轉天數，若 [條件] 則需下修評級至 [等級]
- **跌價損失提列比率**：若財報中「存貨跌價損失」科目出現顯著提列（> [金額]），應立即調升警戒級別
- **同業對比**：[同業名單] 存貨週轉天數約在 [區間]，目前 [公司名] [X] 天處於 [位置]

## 📐 數據採集清單（FinMind API）

### 資產負債表（balance_sheet）
| 科目 | 說明 |
|:---|:---|
| `Inventories` | 存貨總額 |
| `CashAndCashEquivalents` | 現金及約當現金 |
| `ShorttermBorrowings` | 短期借款 |
| `LongtermBorrowings` | 長期借款 |
| `AccountsReceivableNet` | 應收帳款淨額 |
| `CurrentAssets` | 流動資產合計 |
| `CurrentLiabilities` | 流動負債合計 |
| `BondsPayable` | 應付公司債 |
| `TotalAssets` | 資產總額 |

### 損益表（finance / revenue）
| 科目 | 說明 |
|:---|:---|
| `revenue`（月營收） | 用於計算年度營收總和 |
| 或 `Revenue`（年報） | 直接取用年度營收 |

### 現金流量表（cash_flow）
| 科目 | 說明 |
|:---|:---|
| `CashFlowsFromOperatingActivities` | 營業活動之淨現金流入 |
| `PayTheInterest` | 支付之利息 |
| `PropertyAndPlantAndEquipment` | 取得不動產、廠房及設備（用於自由現金流計算） |

### 計算公式
```python
# 存貨週轉率 = 年度營收 / 平均存貨（當季+上季）/ 2
inventory_turnover = annual_revenue / ((current_inventory + prior_inventory) / 2)

# 存貨週轉天數 = 365 / 存貨週轉率
inventory_days = 365 / inventory_turnover

# 流動比率 = 流動資產 / 流動負債
current_ratio = current_assets / current_liabilities

# 速動比率 =（現金 + 應收帳款 + 短期投資）/ 流動負債
quick_ratio = (cash + receivables + short_term_investments) / current_liabilities

# 現金/有息負債覆蓋比
cash_to_debt = cash / (short_term_borrowing + long_term_borrowing + bonds_payable)

# 利息保障倍數
interest_coverage = pre_tax_income / interest_expense

# 自由現金流
fcf = operating_cash_flow - capex
```

## 📋 快速檢查清單（Checklist）

- [ ] 存貨金額是否創近 N 季新高？
- [ ] 單季 QoQ 變動是否超過 ±15%？原因為何？
- [ ] 存貨增速 vs 營收增速比率是否 > 1.5？
- [ ] 週轉天數 vs 同業中位數偏差是否 > 30%？
- [ ] 流動比率是否 > 200%？速動比是否 > 150%？
- [ ] 現金/有息負債覆蓋比是否 > 3x？
- [ ] 負債比是否 < 50%？利息保障倍數是否 > 10x？
- [ ] 是否有明確的備貨原因（訂單/法說會確認）？
- [ ] 跌價損失提列是否異常？

## 📚 實證案例參考

| 公司 | 存貨週轉天數 | 流動比率 | 現金/有息負債 | 評級 |
|:---:|:---:|:---:|:---:|:---:|
| 良維（6290） | 84 天（穩定） | 307% | 11.2x（速動覆蓋） | A（良） |
| 瀚荃（8103） | 68–87 天（波動） | 287% | 4.96x | A（良） |

**關鍵差異：**
- **良維**：週轉天數極度穩定（三年鎖定 84 天），但存貨絕對值快速攀升（+68.4% vs 營收 +38.1%），風險在於「增速不對稱」
- **瀚荃**：週轉天數波動較大（68–87 天），2025Q4 單季跳增 +27%，但現金防禦牆更厚（24.72 億 vs 良維 15.91 億）
