# F18: 估值與決策綜合判斷框架

> **適用範圍**：所有上市櫃公司（跨產業通用分析模式）
> **核心問題**：綜合潛在風險、未來願景與籌碼面，公司當前的合理估值落於何種區間？本益比河流圖是否具備安全邊際？
> **最後更新**：2026-04-25（基於良維 6290、瀚荃 8103 實例提煉）

---

## 🏗️ 框架總覽：五層估值金字塔

```
第5層 ── 決策執行建議（操作等級 + 買區 + 停損）
第4層 ── 評價位階對位（PE/PBR/DY 當前 vs 歷史）
第3層 ── 願景真偽審核（敘事是否有財務數據支撐）
第2層 ── 風險壓力測試（多情境 EPS × PE → 目標價區間）
第1層 ── 數據基石（股價、EPS、營收、籌碼面）
```

**核心原則：好公司 ≠ 好價格。安全邊際是唯一的護城河。**

---

## 第一層：數據基石（Data Foundation）

### 必填基礎數據

| 項目 | 來源 | 用途 |
|:---|---|---|
| 當前股價 | FinMind price API | 估值計算基準 |
| 歷史 EPS（近3-5年） | FinMind finance API | PE 計算、成長率 |
| 歷史 PER/PBR | FinMind PER/PBR API | 估值百分位、河流圖 |
| 月營收年增率（近12-24個月） | FinMind revenue API | 成長動能確認 |
| 法人買賣超（近3-6個月） | FinMind institutional API | 籌碼面趨勢 |
| 股權分散表（大戶佔比） | FinMind backer_holding API | 籌碼集中度 |
| 現金股利歷史 | FinMind dividend API | 殖利率、股東回報 |

### 基礎計算

```
歷史 PE 區間 = 各期股價 ÷ 對應年度 EPS
合理 PE 區間 = 歷史中位數 ± 成長溢價調整
PEG = 當前 PE ÷ EPS 年複合成長率 (CAGR)
估值百分位 = 當前 PE 在歷史區間中的排名位置
```

## 第二層：風險壓力測試（Risk Stress Test）

### 五情境模型

| 情境 | 假設 EPS | 適用 PE | 目標價公式 | 使用時機 |
|:---|---:|---:|---|---|
| 激進樂觀 | 2026E EPS（+20%~30%） | 歷史高點 + 溢價 (25-30x) | EPS × PE | 市場共識極度樂觀時 |
| 穩健（合理估值上緣） | 2026E EPS（+15%~20%） | 歷史高點 (20-25x) | EPS × PE | 成長動能確認中 |
| 中性（歷史中位數） | 2025A EPS | 歷史中位數 (14-18x) | EPS × PE | 基準情境 |
| 悲觀（成長減速） | 2026E EPS（-5%~10%） | 歷史低點偏上 (12-14x) | EPS × PE | 成長動能放緩時 |
| 極端保守 | 2026E EPS（-15%~20%） | 歷史最低點 (10-12x) | EPS × PE | 黑天鵝情境 |

### 壓力測試輸出格式

```markdown
| 情境 | 假設 EPS | 適用 PE | 目標價 | 距現價跌幅 |
|:---|---:|---:|---:|---:|
| 激進樂觀 | X.XX | XXx | XXX元 | -XX% |
| 穩健（合理估值上緣） | X.XX | XXx | XXX元 | -XX% |
| 中性（歷史中位數） | X.XX | XXx | XXX元 | -XX% |
| 悲觀（成長減速） | X.XX | XXx | XXX元 | -XX% |
| 極端保守 | X.XX | XXx | XXX元 | -XX% |
```

### 安全邊際判定規則

| 當前價格 vs 合理區間上緣 | 判定 | 操作指引 |
|:---|---:|---|
| 低於合理區間下緣（-20%以下） | ✅ 具安全邊際 | 分批建倉 |
| 落在合理區間內（±10%） | ⚠️ 中性估值 | 觀望，等待回檔 |
| 高於合理區間上緣（+10%以上） | ⚠️ 溢價估值 | 逢高減碼 |
| 高於合理區間上緣（+30%以上） | ❌ 極端溢價 | 不宜追價，停損防線啟動 |

## 第三層：願景真偽審核（Vision Authenticity Audit）

### 審核框架：四維度驗證

| 維度 | 檢查項目 | 判定標準 |
|:---|---|---|
| **成長動能** | 新產品/新市場佔比趨勢 | 有數據支撐（占比提升+營收貢獻）→ ✅；空洞敘事 → ❌ |
| **資本支出** | CapEx 是否與願景匹配 | 實質投資（廠房、設備、研發）→ ✅；無實際投入 → ❌ |
| **獲利能力** | 毛利率/營業利率趨勢 | 產品組合升級帶動毛利提升 → ✅；毛利持平或下降 → ⚠️ |
| **管理執行力** | 計畫 vs 實際達成率 | 按計畫執行（減資、擴產、股利）→ ✅；屢次跳票 → ❌ |

### 願景評級

| 評級 | 定義 | 估值影響 |
|:---:|---|---|
| **A（真實）** | 四維度 ≥3✅，有明確財務數據支撐 | 可給予歷史中位數 +2~5x 溢價 |
| **B（部分真實）** | 二維度✅，一維度⚠️，成長動能部分確認 | 適用歷史中位數，不給溢價 |
| **C（空洞敘事）** | 零或一維度✅，無實質財務支撐 | 應以歷史低點估值，甚至折價 |

### 關鍵問題：「這是便宜的理由，還是壞掉的真相？」

- **便宜的理由**：景氣循環+成長週期，逢回檔可買
- **壞掉的真相**：結構性衰退，估值再低也不碰

## 第四層：評價位階對位（Valuation Anchoring）

### 三指標交叉驗證

| 估值指標 | 當前值 | 歷史中位數 | 溢價/折價幅度 | 解讀 |
|:---|---:|---:|---:|---|
| 本益比 (PE) | — | — | — | 成長股核心指標 |
| 股價淨值比 (PBR) | — | — | — | 資產型公司參考 |
| 殖利率 (DY) | — | — | — | 收息型公司參考 |

### 估值百分位計算

```
PER 百分位 = (當前 PER - 歷史最低 PER) ÷ (歷史最高 PER - 歷史最低 PER) × 100
PBR 百分位 = (當前 PBR - 歷史最低 PBR) ÷ (歷史最高 PBR - 歷史最低 PBR) × 100
```

### 百分位解讀

| 百分位區間 | 估值狀態 | 操作建議 |
|:---:|---|---|
| 0-20th | 極端低估 | 積極建倉區 |
| 20-50th | 合理偏低 | 分批布局區 |
| 50-80th | 合理偏高 | 觀望區 |
| 80-100th | 極端高估 | 逢高減碼區 |

### 漲幅驅動因子拆解（進階）

```
總漲幅 = 基本面貢獻 + 估值擴張/壓縮貢獻

例：股價從 A → B
- 基本面成長貢獻 = (EPS_B - EPS_A) ÷ EPS_A × 當前 PE
- 估值擴張貢獻 = (PE_B - PE_A) ÷ PE_A × 當前股價
- 驗證：基本面貢獻 + 估值擴張貢獻 ≈ 總漲幅（誤差 ±5%）
```

**核心洞察**：若估值擴張貢獻 > 總漲幅的 40%，則回調風險大於上行空間。

## 第五層：決策執行建議（Action Plan）

### 操作等級

| 等級 | 定義 | 適用情境 |
|:---:|---|---|
| **A（買進）** | 具安全邊際 + 願景真實 + 籌碼正面 | 分批建倉，目標價區間內 |
| **B+（持有）** | 合理估值 + 願景部分真實 | 已持有者續抱，空手者觀望 |
| **B（持平）** | 溢價估值 + 願景真實但已定價 | 逢高減碼，等待回檔 |
| **B-（觀望）** | 極端溢價 + 願景部分真實 | 不宜追價，停損防線啟動 |
| **C（賣出）** | 極端溢價 + 願景空洞 + 籌碼惡化 | 執行停損，尋找替代標的 |

### 買進區間設定規則

| 買區等級 | PE 對應 | 安全邊際 | 適用條件 |
|:---:|---:|---:|---|
| **第一買區** | 歷史高點附近 (20-25x) | 低 | 需營收確認加速信號 + 籌碼正面 |
| **第二買區** | 歷史中位數偏上 (15-20x) | 中 | 具基本安全邊際，可分批布局 |
| **第三買區** | 歷史中位數以下 (10-15x) | 高 | 成長減速確認後的價值回歸點 |

### 停損防線設定

| 觸發條件 | 操作 | 理由 |
|:---|---|---|
| 跌破第一買區下緣（-15~20%） | 減碼 30-50% | 技術面轉弱，先降曝險 |
| 跌破第二買區下緣（-25~35%） | 執行停損 | 成長預期遭破壞，估值修正確認 |
| 跌破第三買區下緣（-40%+） | 全部出場 | 結構性問題，尋找替代標的 |

### 關鍵指標監控清單模板

```markdown
| 指標 | 當前值 | 警戒線（下） | 警示線（上） |
|:---|---:|:---:|:---:|
| PER | — | <15x（可布局） | >25x（過熱） |
| 月營收年增率 | — | <5%（成長停滯） | — |
| 毛利率(季) | — | <XX%（產品組合惡化） | — |
| 新業務佔比 | — | 持平或下降 | — |
| 千張大戶佔比 | — | <XX%（強勢手撤退） | — |
| 外資單週淨買賣超 | — | < -XXX萬張（結構性撤退） | — |
```

## 🔧 實作檢查清單（Checklist）

每次執行 F18 審計時，依序完成：

- [ ] **第0步**：收集基礎數據（股價、EPS、營收、籌碼面）
- [ ] **第1步**：計算歷史 PE 區間（3-5年）與當前估值百分位
- [ ] **第2步**：執行五情境壓力測試，繪製合理估值區間
- [ ] **第3步**：四維度願景審核（成長動能、CapEx、毛利率、管理執行力）
- [ ] **第4步**：三指標交叉驗證（PE、PBR、DY）
- [ ] **第5步**：漲幅驅動因子拆解（基本面 vs 估值擴張）
- [ ] **第6步**：綜合評級判定（風險 × 估值 × 願景）
- [ ] **第7步**：輸出操作等級、買區建議、停損防線
- [ ] **第8步**：建立關鍵指標監控清單
