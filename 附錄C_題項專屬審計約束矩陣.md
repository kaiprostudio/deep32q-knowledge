# 附錄 C：題項專屬審計約束矩陣

> 本附錄為 03_審計專家協議 3.6 節的參考資料。各題項的關鍵數據與校準路徑如下：

### 財務組 (F01-F18)

| 題號 | 關鍵數據與校準路徑 |
| :--- | :--- |
| **F01-F03** | 強制進行 3 年財務對比；F03 須計算「訂單能見天數」：`CurrentContractLiabilities / AvgRevenuePerDay`。 |
| **F04** | [業外純度穿透]：檢測處分資產是否掩蓋本業虧損；核對子公司認列方法。 |
| **F05-F07** | [存貨與應收專項] 計算「營運週期 (CCC)」；F07 執行「階差分析」。 |
| **F08** | 用「在建工程/預付設備款」驗證資本支出進度。 |
| **F09-F10** | 檢索「固定成本/折舊/薪資」佔比，執行「經營槓桿拐點」分析。 |
| **F11** | 辨識資產處分是「戰略轉型」還是「祖產求生」。 |
| **F12** | 計算各年度「研發產出比 (Revenue/R&D)」。 |
| **F13** | [股利覆蓋率]：分析配息是否來自「借貸」或「處分收益」。 |
| **F14** | 分析負債結構與利息保障倍數趨勢。 |
| **F15** | 檢索「內部人/大股東持股異動」，偵測誠信風險。 |
| **F16-F17** | 調用「股權分散表」；分析「增資成本價格帶」與「週變動鎖碼特徵」。 |
| **F18** | 綜合評估，給出最終買賣建議。 |

### 經營組 (G01-G11)

| 題號 | 關鍵數據與校準路徑 |
| :--- | :--- |
| **G01** | 資料時間一致性校準，確保歷史沿革與現狀對齊。 |
| **G02** | 深度分析「法人代表更換」背景。 |
| **G03** | 獲取核心人員離職率資訊。 |
| **G04-G06** | [產品與通路校準]：比對三年內「主要客戶佔比」變動；G04 含市佔率校準。 |
| **G07** | [研發實質性]：核對「研發費用」與「研發成果」之變現率。 |
| **G08** | [市場與風險敞口]：檢索「分部資訊」獲取區域營收。 |
| **G09** | [供應鏈安全性]：導出「前十大客戶/供應商」清單，計算「集中度風險指數」；含合作夥伴戰略深度。 |
| **G10** | [護城河]：分析產業壁壘與競爭優勢持久性。 |
| **G11** | [戰略執行力預測]：核對「購置固定資產」支出，驗證擴產宣稱；含新業務展望。 |
