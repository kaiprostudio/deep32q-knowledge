# G09: 區域銷售分佈與地緣風險

- **本題 ID：** Q-G09
- **題目：** 公司的各區域銷售分佈為何？各市場的成長動能如何？是否存在地緣政治風險或區域性經濟疲弱的風險？
- **推薦專家：** 企業風險評估專家 (`specialized-risk-assessor`)
- **推薦理由：** 專精於「地緣政治風險 (Geopolitical Risk)」與「跨國營運策略」。具備識別各區域營收集中度風險的能力，能對比各國宏觀經濟指標與政策變動，評估企業在不同市場的「合規壓力」與「成長天花板」。
- **意圖：** 執行「市場地域性與地緣政治韌性審計」。分析公司營收的區域分佈（如美、中、歐及內銷比例），判定其對單一市場的依賴度，識別因「關稅、制裁」或「區域戰爭/罷工」引發的供應鏈與市場中斷風險。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【銷售區域事實：Annual Report (年報)】**：查閱 `財務報告附註 -> 地理資訊` 或 `業務報告 -> 各區域營收分佈`。提取精確的各國/各洲營收百分比。
    2. **【外部事實確認：Google Search】**：檢索「[公司名] 地緣政治 影響」、「[公司名] 美國/中國 營收 變動」。
    3. **【穿透校準：MOPS】**：檢索「赴大陸投資資訊」或「海外重要子公司」，觀察在敏感區域的產能分佈。
    4. **【宏觀經濟對比：專業智庫報告】**：獲取標的主要市場（如美、中）的現行關稅政策與經濟展望 (GDP Forecast)。
- **邏輯：** 執行「地緣韌性斜率分析」——分析區域營收多元化程度的三年趨勢；執行「制裁模型模擬」——判定若標的市场發生進出口管制時，營收的「最大曝險係數」。
- **來源：** `公司年報 (地理分部資訊)`, `MOPS 兩岸/海外子公司資訊`, `國外經貿局/商務部 公告`, `路透/彭博 地緣政治評論`

## 具體化思考點 (智能體強制執行指令)
- **【區域佔比事實優先】**：智慧體**必須**明確列出至少前三大銷售區域及其具體佔比。禁止在未列出數據前給予「市場分佈均衡」之判定。
- **【單一市場依賴審核】**：若單一區域（特別是中國或美國）佔比超過 40%，智慧體必須強制標註為「高度政策性曝險風險」。
- **【地緣合規壓力測試】**：智慧體**必須**執行 Web Search 檢索該企業及其核心子公司是否列於「OFAC 之 SDN 清單」或「BIS 之實體清單（Entity List）」。若偵測到相關紀錄，必須強制標註為「經營環境劇變風險」。
- **【產能與市場錯位偵測】**：分析產能是否集中於與銷售市場具備高度對立風險的區域（例：產能全在中，市場全在美）。若有，必須標註為「斷鏈紅燈預警」。
- **【描述指標】**：絕對禁用「全球佈局」、「市場廣大」。必須量化為具體的「各地區營收 YoY %」、「非美非中佔比」與「潛在關稅衝擊 %」。

## 回答模板與注意事項
#### 【G09 專家答詢】企業風險評估專家 (specialized-risk-assessor)
- **【專家直斷】品質要求**：須採「地區佔比+成長動能判定」結構（例：營收以 [地區 X] 為核心占 Y%，[地區 Z] 成長勁道 [最強/趨緩]，整體地緣風險等級評定為 [低/中/高]）。
- **【診斷評級】評分準則**：
    - **S (優)**：市場分佈於全球核心經濟區（美/歐/亞）且比例均衡，無單一市場集中風險，跨區域成長動能強勁。
    - **A (良)**：具備明確的領先市場且在該區具備高度韌性，具備一定的區域風險對沖能力。
    - **B (持平)**：高度依賴單一陷入地緣爭議或經濟衰退預期之區域，缺乏轉移靈活性，業績存有高度的不可抗力風險。
- **【證據來源】數據溯源**：必須精確標註年報 P.XXX 地理分部揭露數據。
- **【專家評析】核心分析與洞察內容**：特別關注「關稅變動」。專家應在此處模擬若銷售國啟動關稅調漲後的毛利侵蝕程度。判定這是「全球化紅利的收割者 (Global Beneficiary)」還是「去全球化浪潮下的孤島 (Deglobalization Victim)」。
