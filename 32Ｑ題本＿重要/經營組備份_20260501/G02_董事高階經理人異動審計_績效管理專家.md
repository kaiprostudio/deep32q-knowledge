# G02: 董事高階經理人異動審計

- **本題 ID：** Q-G02
- **題目：** 董事會與高階經理人近三年是否有非正常異動？這對公司目前經營決策與管理穩定度有何影響？
- **推薦專家：** 績效管理專家 (`hr-performance-reviewer`)
- **推薦理由：** 專精於「組織行為」與「治理結構審計」。具備識別權力中樞異動異常訊號的能力，能區分「正常退休/階隊接班」與「內部鬥爭/責任規避」。能從人事異動中讀取企業獲利轉折點或誠信紅線。
- **意圖：** 執行「組織權力穩定性與治理誠信審計」。分析核心經營者的流動，判定是否存在「權力真空」或「管理路徑大規模偏移」，識別因核心人員撤離而引發的經營危機，為公司的「執行穩定性」提供預警。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心事實來源：MOPS (公開資訊觀測站)】**：檢索「重大訊息 -> 董監事及高階經理人異動」。提取精確的異動日期與官方理由（如：職務調整、屆期改選、個人生涯規劃）。
    2. **【穿透校準：Annual Report (年報)】**：查閱「公司治理報告 -> 董事及經理人資料」。比對歷年名單，識別是否存在重要部門（如 CFO, CTO, 稽核主管）的頻繁撤換。
    3. **【外部驗證：Web Search】**：檢索「[公司名] 高管 離職 真相」、「[公司名] 家族內鬥」。透過第三方媒體報導識別是否有檯面下的衝突。
- **邏輯：** 執行「人事異動頻率壓力測試」——若一年內超過三位核心經理人（含財務主管）異動，判定為「治理失靈」；執行「接班順位算則」——分析繼任者背景（內部晉升 vs 外部空降）之決策延續性。
- **來源：** `MOPS 重大訊息 (官定證據)`, `年度報告 (治理專章)`, `商業雜誌與深度特寫`, `LinkedIn (追蹤離職高管去向)`

## 具體化思考點 (智能體強制執行指令)
- **【核心關鍵位階審核】**：智慧體**必須**重點鎖定：董事長、總經理、財務主管、稽核主管、研發主管。若上述位置發生異動，禁止僅以「職務調整」帶過，必須深挖具體背景。
- **【非正常時點偵測】**：分析異動是否發生在「財報發布前夕」或「重大收購案執行期」。若人事異動與股價暴跌或重訊更正同步，必須強制標註為「誠信崩潰警訊」。
- **【接班斷代風險】**：若離職者為創辦人或核心技術領袖且由「非相關領域者」接任，智慧體必須標註為「文化斷代與技術流失風險」。
- **【描述指標】**：絕對禁用「人事穩定」、「平穩過度」。必須量化為具體的「異動率 %」、「平均留任年資（高管級）」與「繼任者磨合天數」。

## 回答模板與注意事項
#### 【G02 專家答詢】績效管理專家 (hr-performance-reviewer)
- **【專家直斷】品質要求**：須採「異動性質+管理衝擊」結構（例：核心經營層穩定，近期 CFO 異動為 [正常接班/內部排擠]，對財務策略 [無影響/有風險]；目前權力架構 [平衡/不穩]）。
- **【診斷評級】評分準則**：
    - **S (優)**：經營層長期穩定，異動均為計畫內接班，且新任者具備優異的資歷與一致的策略方向。
    - **A (良)**：具備局部的高管異動，但屬於正常的績效汰換或組織擴張所需，具備成熟的接班體系。
    - **B (持平)**：財務、稽核或技術核心頻繁撤換，且異動理由模糊，顯示團隊陷入治理風暴或存在潛在財務爆雷掩飾，評級為「重大治理風險」。
- **【證據來源】數據溯源**：必須精確標註 MOPS 重訊編號、年報 P.XXX 治理架構表。
- **【專家評析】核心分析與洞察內容**：特別關注「稽核主管」與「財務主管」的異動。這兩者的同時離職通常被視為「財報即將爆雷」或「內控失效」的全球通用紅燈訊號，專家應在此處進行最嚴厲的風險評估。
