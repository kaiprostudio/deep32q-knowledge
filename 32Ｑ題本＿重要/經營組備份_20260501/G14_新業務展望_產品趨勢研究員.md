# G14: 新業務展望

- **本題 ID：** Q-G14
- **題目：** 公司在最近三年中是否有增加新的業務展望或第二成長曲線？其技術儲備與市場佈局進度為何？
- **推薦專家：** 產品趨勢研究員 (`product-trend-researcher`)
- **推薦理由：** 專注於行業趨勢分析與技術前瞻。擅長辨識「第二成長曲線」的真實爆發力，能從全球產業演進的角度，判定公司新增業務是順應未來 3-5 年的趨勢風口，還是進入過度飽和的夕陽市場，識別「無效轉型」風險。
- **意圖：** 鑑定業務邊界的擴張能力與未來獲利天花板。判定其新業務是否具備實質的競爭門檻，識別其在「研發期」、「樣品送審期」或「放量期」的具體位置。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【趨勢對位核心：IDC / Gartner / 專業智庫報告】**：獲取標的新業務所屬產業的「未來年複合成長率 (CAGR)」與市場規模 (TAM)。
    2. **【展望事實來源：Annual Report (年報)】**：必須查閱 `業務概況 -> 未來計畫`。提取管理層對新產品線或新市場的具體描述。
    3. **【穿透校準：MOPS】**：檢索「重大收購」、「董事會決議重要資本支出」或「重要契約簽訂」公告。
    4. **【外部驗證：產品官方頁面 (New Products Launch)】**：核實新產品是否已正式掛載官網或獲得相關技術認證。
- **邏輯：** 執行「趨勢對位審計」——分析新業務是否具備較高的成長增速，並驗證標的公司是否具備切入該賽道的技術門檻。
- **來源：** `公司法說會簡報 (未來佈局)`, `產業趨勢報告 (Trend Reports)`, `專業機構市調數據`, `官方網站新產品分類`

## 具體化思考點 (智能體強制執行指令)
- **【賽道趨勢對位】**：智慧體**必須**檢索該新業務所處產業的未來前景。針對新賽道，**強制列出其未來 5 年之市場規模 (TAM) 預測與年複合成長率 (CAGR)**。若新業務進入的是紅海競爭或夕陽產業，必須在評級中給予警示。
- **【早期信號捕獲與 Design-win】**：智慧體**必須**檢索標的公司在新領域是否已取得「Design-win (設計贏取)」或「官方認證/樣品測試通過」之實質信號。若僅停留在「規劃階段」而無具體料號或驗證公告，必須下調評級。
- **【描述指標】**：絕對禁用「轉型成功」、「展望樂觀」。必須量化為具體的「新業務預計貢獻佔比」、「目標市場規模 (USD)」與「技術認證取得進度」。

## 回答模板與注意事項
#### 【G14 專家答詢】產品趨勢研究員 (product-trend-researcher)
- **【專家直斷】品質要求**：直接判定其新業務展望是「符合大勢、具備爆發潛力」、「合理補強、但增速有限」還是「錯誤轉型、進入紅海」。
- **【診斷評級】評分準則**：
    - **S (優)**：新業務位於高速成長賽道，且公司具備顯著的切入優勢（技術或通路），已有指標性訂單或 Design-win 實績。
    - **A (良)**：戰略方向正確，處於市場回報初期，技術研發進度符合預期，具備穩健的擴張潛力。
    - **B (持平)**：新業務與現有趨勢背離，或進入門檻極高且公司缺乏核心競爭力，轉型進度滯後且缺乏明確亮點。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力，並給出物理對策。
- **【證據來源】數據溯源**：列出市場研究報告數據、新業務公告、或法說會具體展望頁碼。
- **【專家建議】注意事項**：提示投資人關注「該技術領域是否可能被新科技迅速取代」的顛覆性風險。
