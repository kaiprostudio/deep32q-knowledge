# G13: 未來發展與戰略規劃

- **本題 ID：** Q-G13
- **題目：** 管理層給出的未來 3-5 年發展規劃與戰略目標為何？是否有具體的擴產或併購計畫？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 擅長執行「資本配置效率分析 (Capital Allocation Efficiency)」。能透過現金流量表與資產負債表，驗證管理層的戰略目標是否有對應的資金投入 (CapEx) 與產能對陣，判定其成長藍圖的可信度。
- **意圖：** 執行「成長動能實質性與經營層執行決心審計」。判定投資案的「未來預期溢價」，識別「畫大餅型願景」與「實質擴張型戰略」之差異點。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【行動實測基石：FinMind】**：調用 `TaiwanStockCashFlowsStatement` 提取「投資活動現金流 -> 購置固定資產 (CapEx)」。比對營收成長預期。
    2. **【願景獲取來源：Annual Report (年報)】**：必須查閱 `致股東報告書 -> 未來展望` 或 `管理層討論與分析 (MD&A)`。
    3. **【穿透校準：MOPS】**：檢索「重大收購案」、「處分重大資產」或「董事會決議擴產」公告。
    4. **【外部驗證：法人說明會 (Earnings Calls)】**：查閱經營層對未來展望的具體口徑與分析師問答。
- **邏輯：** 執行「行與言一致性測試」——真正的發展規劃必須體現在「資本支出 (CapEx)」的增長上；若僅有願景而無資金流出，應判定為「宣傳性規劃」。
- **來源：** `公司致股東報告書`, `FinMind (資產支出數據)`, `重大訊息公告 (MOPS)`, `法說會展望簡報`

## 具體化思考點 (智能體強制執行指令)
- **【CapEx 真實性檢核】**：智慧體**必須**檢索最近 4 季的「購置固定資產」金額。若管理層宣稱要大舉擴產但在財報上看不到對應支出，必須標註為「戰略執行度存疑」。
- **【戰略連貫性審計】**：搜尋該公司兩年前的法說會紀錄，比對當時承諾的「戰略目標」與今日的「達成進度」。若目標頻繁更換且無一達成，應下修其管理信用。
- **【併購/擴產動機判定】**：分析併購案是為了「擴大市場份額」還是「補足技術缺口」。
- **【描述指標】**：絕對禁用「前景光明」、「計畫宏大」。必須量化為具體的「未來產能擴張 %」、「預計投資回報率 (ROA/ROE)」與「資產支出率 (CapEx/Sales)」。

## 回答模板與注意事項
#### 【G13 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：直接判定其規劃是「路徑清晰、具備財力支持」、「過於激進、財務壓力大」、還是「目標模糊、具備畫大餅嫌疑」。
- **【診斷評級】評分準則**：
    - **S (優)**：戰略路徑清晰且已有明確的資本支出支撐，與核心競爭力高度契合，具備極高的達成概率。
    - **A (良)**：具備明確成長方向與執行計畫，但需觀察資金回收週期與市場吸納能力，財務結構尚可支撐。
    - **B (持平)**：目標過於宏大但缺乏具體數據支撐，或與目前財務狀況存在明顯脫節，執行風險極高。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力，並給出物理對策。
- **【證據來源】數據溯源**：必須精確標註 FinMind 投資現流數據、年報展望段落、或重大訊息公告字號。
- **【專家建議】注意事項**：提示投資人關注「擴產後的產能利用率」是否可能因市場飽和而導致資產減損風險。
