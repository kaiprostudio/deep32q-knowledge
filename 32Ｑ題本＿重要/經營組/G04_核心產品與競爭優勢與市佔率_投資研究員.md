# G04: 核心產品、競爭優勢與市佔率

- **本題 ID：** Q-G04
- **題目：** 公司的核心產品與服務為何？對比同業具備何種技術或成本優勢？在當前競爭環境中的行業佔有率為何？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 專精於「產品定位與市場競爭力分析」。能從產品結構、技術優勢、市佔率與同業對比中，判定公司在產業鏈中的真實位階與競爭力持久性。
- **意圖：** 執行「產品競爭力與市場位階審計」。從產品結構（本質是什麼）、技術優勢（為何能贏）、市佔率（贏了多少）三個維度，判定公司的競爭力是「結構性優勢」還是「暫時性領先」。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockRevenue` 營收結構、`TaiwanStockFinancialStatements` 毛利率趨勢。
    2. **【產品定位來源：Annual Report (年報)】**：查閱 `業務概況 -> 核心產品` 與 `產業競爭分析`。
    3. **【市佔率校準：同業財報】**：透過 FinMind 或搜尋取得同業營收，計算相對市佔。
    4. **【外部驗證：搜尋】**：檢索同業動態、產業報告、券商分析。
- **邏輯：** 執行「三維競爭力矩陣」——(1)產品是什麼(結構)、(2)為何能贏(優勢)、(3)贏了多少(市佔)。三者缺一不可。
- **來源：** `FinMind (營收與財報)`, `年報 (產品與競爭分析)`, `同業財報 (市佔對比)`, `產業報告`

## 具體化思考點 (智能體強制執行指令)
- **【產品結構拆解】**：必須量化各產品線佔比與成長趨勢，禁止模糊描述「產品多元」。
- **【技術優勢穿透】**：必須具體指出「護城河」的物理載體（專利？認證？規模？客戶黏性？），禁止空泛描述「技術領先」。
- **【市佔率量化】**：必須計算公司在整體市場與細分市場的市佔率，與同業進行營收/毛利/成長率對比。
- **【相對位階對比】**：列出至少 2-3 家同業的營收、毛利率、成長率，判定公司在產業中的真實排位。
- **【描述指標】**：絕對禁用「產品優質」、「市佔領先」。必須量化為具體的「產品線佔比 %」、「毛利率階差 ppt」、「市佔率 %」。

## 回答模板與注意事項
#### 【G04 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：須採「產品結構 + 技術優勢 + 市佔位階」三段式結構。
- **【診斷評級】評分準則**：
    - **S (優)**：核心產品具備不可替代性（技術壁壘 + 客戶黏性），市佔率穩步提升，毛利率顯著優於同業。
    - **A (良)**：產品具備明確優勢，市佔率穩定或小幅成長，毛利率維持在同業前段。
    - **B (持平)**：產品優勢不明顯或正在被侵蝕，市佔率停滯或下滑，毛利率面臨壓力。
- **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐。
- **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力。
- **【證據來源】數據溯源**：必須精確標註數據來源（FinMind/年報/搜尋）。
