# F06: 存貨與債務穩定度審計

- **本題 ID：** Q-F06
- **題目：** 存貨金額變動趨勢為何？流動性資產是否足以覆蓋短期與長期債務？
- **推薦專家：** 財務分析師 (`finance-financial-analyst`)
- **推薦理由：** 專精於「流動性風險分析 (Liquidity Risk Analysis)」與「資產負債管理」。能從存貨組成（原料/半成品/製成品）中讀取訂單去化動態，並精算償債倍數，是企業防禦力的終極評審。
- **意圖：** 執行「資產流動性與償債能力審計」。透過分析存貨水位與債務結構，判定公司是否面臨「庫存堆積導致跌價損失」或「流動性斷鏈」風險。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockBalanceSheet` 提取 `Inventories` (存貨)、`ShortTermBorrowings` (短期借款)、`LongTermBorrowings` (長期借款) 與 `QuickAssets` (速動資產)。
    2. **【穿透校準：Annual Report (年報)】**：查閱 `財務報告附註 -> 存貨組成` 與 `跌價損失準備`。
    3. **【外部事實檢索：產業景氣循環】**：搜尋該產業（如自行車、面板、半導體）目前是否處於「去庫存週期」。
- **邏輯：** 執行「存貨/債務交叉覆蓋率測試」——若存貨週轉天數拉長且短期借款增加，判定為「庫存造成的現金流黑洞」；執行「跌價風險試算」。
- **來源：** `FinMind (資產/債務)`, `年度報告 (存貨附註)`, `財經新聞 (產業去庫存動態)`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **跑工具**：`python3 財務分析工具.py inventory_check [股號]` → 取得庫存週轉、DSO、DPO、現金循環
2. **取資產負債表**：`python3 數據採集器.py balance_sheet [股號]` → 抓 `CashAndCashEquivalents`、`ShortTermBorrowings`、`LongTermBorrowings`
3. **計算流動性**：流動比率 = 流動資產/流動負債；速動比率 = (流動資產-存貨)/流動負債
4. **判讀**：流動比率 >200% = 優；100-200% = 正常；<100% = 警戒
5. **年報補充**：用 `pdf` 工具提取「存貨明細」和「借款附註」
6. **撰寫報告**：依下方回答模板結構產出

## 具體化思考點 (智能體強制執行指令)
- **【存貨組成分析】**：智慧體**必須**列出最近三季的「存貨總額」。若「製成品」佔比異常升高而營收下滑，必須判定為「產品滯銷」。
- **【償債壓力測試】**：測算「(現金 + 速動資產) / 短期債務」。若比值 < 1，且存貨週轉緩慢，必須發出「流動性斷鏈」之警訊。
- **【跌價損失預警】**：若存貨天數超過同業平均 30% 以上，必須強制要求智體評估其「跌價損失提列」是否足以覆蓋潛在價值衰減。
- **【Ticker 校準】**：使用 yfinance 時，必須遵循 `.TW` (上市) 或 `.TWO` (上櫃) 規範，嚴禁輸入純數字。
- **【描述約束】**：絕對禁用「負債平穩」、「存貨充足」。必須量化為「存貨週轉率」與「流動比/速動比」。

## 回答模板與注意事項
#### 【F06 專家答詢】財務分析師 (finance-financial-analyst)
- **【專家直斷】品質要求**：須採「資產流動性+債務壓力」結構（例：存貨 YOY 增加 X%，週轉天數拉長，雖 [短期債務] 有現金覆蓋，但 [庫存去化] 慢將導致未來跌價風險，評級為 [警訊]）。
- **【診斷評級】評分準則**：
    - **S (優)**：存貨週轉極快，現金完全覆蓋所有短債，且無長期償債壓力。
    - **A (良)**：存貨水位與業績成長匹配，債務結構合理，利息支用比在安全水位。
    - **B (持平)**：存貨積壓嚴重且債務倍數升高，現金流量難以支撐連續兩季以上的去庫存壓力，具備實質性的資產減值與違約風險。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力，並給出物理對策。
- **【證據來源】數據溯源**：必須精確列出 FinMind 季度資產負債數據與計算得出的流動比率。
