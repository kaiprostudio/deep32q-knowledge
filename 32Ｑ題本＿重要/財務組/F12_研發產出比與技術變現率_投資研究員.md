# F12: 研發產出比與技術變現率

- **本題 ID：** Q-F12
- **題目：** 近三年研發費用 (R&D) 投入與實際營收/獲利產出的關聯性為何？技術轉化效率是否正在改善？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 專精於「技術資本化分析」與智慧財產 (IP) 價值評估。擅長計算「研發產出率 (R&D Yield)」與技術生命週期，識別企業是在進行引領行業的「突破性創新」，還是在為維持競爭力而進行的「防禦性研發燒錢」。
- **意圖：** 執行「技術 ROI 與創新驅動審計」，判定公司研發投資是否實質轉化為毛利率提升或市佔率擴張，識別長期「研發虛耗」與「技術路徑錯誤」風險。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockFinancialStatements` 提取 `ResearchAndDevelopmentExpense` (研發費用)。計算「研發費用率 (R&D Intensity)」。
    2. **【外部事實確認：Google Search】**：檢索「[公司名] 新產品上市」、「[公司名] 專利獲權數量」、「[公司名] 技術領先地位 評論」。
    3. **【穿透校準：Annual Report (年報)】**：查閱「業務概況 -> 研究發展」節，詳列「主要研發成果」與「未來五年開發計畫」之進度與預計量產時間。
- **邏輯：** 執行「技術變現算則」——分析研發費用成長與次年「新產品營收貢獻度」的連動性；執行「研發槓桿測算」分析單位研發支出創造的營業利潤增量。
- **來源：** `FinMind (損益表詳列)`, `公司年報 (研發報告節)`, `智慧財產局專利資料庫`, `產業技術白皮書`

## 具體化思考點 (智能體強制執行指令)
- **【研發事實優先】**：智慧體**必須**列出近三年的「研發費用額」及其佔營收比重。禁止在未列出投入強度前給予「技術優勢」之讚譽。
- **【變現效率偵測】**：智慧體必須對比「研發強度」與「毛利率趨勢」。若研發投入增加但毛利率連續四季下滑，必須強制標註「研發轉化無效」或「技術路徑脫節」風險。
- **【創新含金量審核】**：檢索是否有「顛覆性技術」投產的消息。判定研發是屬於「微幅改良」還是「跨世代升級」。
- **【Ticker 校準】**：使用 yfinance 時，必須遵循 `.TW` (上市) 或 `.TWO` (上櫃) 規範，嚴禁輸入純數字。
- **【描述指標】**：絕對禁用「重視研發」、「成果輝煌」。必須量化為具體的「研發產出倍數 (Revenue/R&D)」與「技術生命週期 (Months)」。

## 回答模板與注意事項
#### 【F12 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：須採「研發強度+轉化鑑定」結構（例：R&D 佔比 X%，帶動 Y% 毛利增長，判定技術變現 [高效/滯後]，技術護城河 [擴張/萎縮]）。
- **【診斷評級】評分準則**：
    - **S (優)**：研發投入精準轉化為市場領先的高毛利產品，研發產出率持續攀升，且具備專利保護牆，技術變現動能極強。
    - **A (良)**：研發投入與成長規模匹配，能穩定推出次世代產品以維持競爭力，技術轉化路徑清晰且獲利穩定。
    - **B (持平)**：陷入「研發黑洞」——研發支出巨大但營收成長停滯，或新產品市場反應冷淡，面臨超重的資本虛耗與技術落後風險。
- **【答詢詳解】解析深度**：依照以下三個維度回答：
    1. **研發槓桿分析 (R&D Leverage Audit)**：分析單位研發投入對營業利益的成長拉動力。
    2. **技術路徑競爭力 (Tech Path Competitiveness)**：對比同業分析該公司在技術規格上的實質領先優勢。
    3. **專利質量與變現 (IP Value Monetization)**：評估企業 IP 組合的變現潛力與法律侵權風險。
- **【證據來源】數據溯源**：必須精確標註 FinMind `financial` 損益表科目與年報 P.XXX 研發專案進度表。
- **【專家評析】核心分析與洞察內容**：特別關注「研發費用資本化」之比例。若公司將大量研發支出列為「無形資產」而非費用，專家應在此處診斷為「財報美化風險」。
