# F13: 資本支出效能與股利防禦

- **本題 ID：** Q-F13
- **題目：** 資本支出 (CapEx) 的投入效能為何？自由現金流 (FCF) 是否足以支撐其股利政策？是否存在「借債配息」之虛耗現象？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 專精於「資本配置 (Capital Allocation)」與「股東回報模型」。擅長分析企業在擴張與分紅之間的平衡藝術，能識別企業是在進行「高回報投資」，還是在進行「無效的資本虛耗」，為長期持有的價值提供安全校準。
- **意圖：** 執行「資本支出回報與財務防禦審計」。透過分析 CapEx 成長與營收/利潤成長的階差，判定擴產效能；透過精算 FCF，判定股利支付的安全性，識別是否存在掏空公司資產以迎合市場分紅預期的風險。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockCashFlowsStatement` 提取 `CapitalExpenditures`、`CashFlowsFromOperatingActivities` 與 `CashDividendPaid`。
    2. **【外部事實確認：優分析/財經媒體 (cnYES/MOPS)】**：獲取歷年「股利發放率 (Payout Ratio)」與「殖利率」趨勢。
    3. **【穿透校準：Annual Report (年報)】**：查閱「現金流量表附註」與「股利政策說明」。
- **邏輯：** 執行「CapEx 成長階差分析」——計算 `(營收成長率 / CapEx 成長率)`；執行「現金流覆蓋測試」——計算 `FCF / 總配息金額`。
- **來源：** `FinMind`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **跑工具**：`python3 財務分析工具.py capex_efficiency [股號]` → 取得 CapEx、FCF
2. **取股利**：`python3 數據採集器.py dividend [股號]` → 抓現金股利
3. **判讀**：FCF/股利 <1 = 借債配息警訊；CapEx/營收比 >20% = 擴張期
4. **年報補充**：用 `pdf` 工具提取「股利政策」
5. **撰寫報告**：依下方回答模板結構產出

⚠️ **常見陷阱**：①答詢詳解 ≥200 字、專家評析 ≥200 字；②空頭視角 ≥2 個；③FCF 要含 CapEx 計算

## 具體化思考點 (智能體強制執行指令)
- **【資本支出效能實測】**：智慧體**必須**列出最近 3 年的 CapEx 金額與次年營收成長的連動性。嚴禁在未計算回報率前給予「積極擴產」之正面評價。
- **【配息含金量審核】**：智慧體必須計算 `自由現金流 (CFO - CapEx)`。若自由現金流為負但仍發放高額股利，必須強制發出「損害長期競爭力」或「借債發薪」之嚴厲警訊。
- **【股利穩定性測試】**：判定在利潤下滑 20% 的情境下，既有的股利政策是否具備「防禦韌性」。
- **【描述指標】**：絕對禁用「配息大方」、「投入巨大」。必須量化為具體的「FCF 覆蓋倍數」、「CapEx 回報係數」與「資產支出率 %」。

## 回答模板與注意事項
#### 【F13 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：須採「支出效能+配息安全性」結構（例：CapEx 回報率為 X，FCF 覆蓋率為 Y，判定 [投資高效/無效]，股利政策屬 [穩健/虛耗] 型態）。
- **【診斷評級】評分準則**：
    - **S (優)**：投資回報顯著（營收增幅高於支出增幅），且 FCF 極其充裕，足以同時支撐擴產與優渥的分紅，資本配置處於「正向循環」極致。
    - **A (良)**：擴產節奏合理，分紅由營運現金流支撐，財務體質健康。
    - **B (持平)**：CapEx 巨大但營收停滯，或分紅高度依賴舉債/處分資產，顯示資本配置失能，長期價值具備「崩塌風險」。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力，並給出物理對策。
- **【證據來源】數據溯源**：必須精確標註 FinMind `cashflow` 數據項與計算過程。
