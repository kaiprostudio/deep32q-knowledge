# F08: 產能利用率與資產擴張

- **本題 ID：** Q-F08
- **題目：** 公司的產能利用率與固定資產擴張率為何？目前處於重資產擴張期 (Heavy Asset) 還是輕資產化轉型？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 專精於「資本支出效率」與「資產擴張動能」。能從硬數據讀懂管理層對未來的信心（生意太好才敢擴產）或恐懼（擴不動、被迫轉型）。其審計視角能確保「業績動能」是真金白銀，而非資產泡沫。
- **意圖：** 判定「業績含金量」與「未來獲利爆發點」——透過交叉比對產能擴張速度與營收/利潤成長速度，識別該公司是處於「爆發式成長前夕之佈局期」、「高效收割期」還是「轉型瓶頸期」。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【第一優先：FinMind】**：調用 `TaiwanStockBalanceSheet` 提取 `PropertyPlantAndEquipment` (固定資產)；調用 `TaiwanStockCashFlowsStatement` 提取 `Depreciation` (折舊費用)。比對近三年成長率。
    2. **【第二優先：專業財經站點】**：轉向 **MOPS (公開資訊觀測站)**、**優分析**、**優分析/各大財經平台**。提取「在建工程 (CIP)」子項明細，確認未來「新增產能」的具體量度。
    3. **【第三優先：公司官網】**：檢索「法說會簡報 (Presentation)」中關於 **稼動率 (Utilization Rate)** 的指引與未來的擴產時間軸。
    4. **【最後手段：Web Search】**：檢索市場對新廠產能被客戶「包下」的報導，驗證生意的真實火熱程度。
- **邏輯：** 執行「獲利成長領先/落後分析」——計算 `營收成長率 / 固定資產成長率`。若系數 > 1.2，代表生意太好，產能供給跟不上需求；執行「產能釋放演習」——推算 CIP 轉固後對次年營收的貢獻邊際。
- **來源：** `FinMind`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **跑工具**：`python3 財務分析工具.py capex_efficiency [股號]` → 取得 CapEx、FCF、CapEx/營收比
2. **取數據**：`python3 數據採集器.py cash_flow [股號]` → 抓 PropertyAndPlantAndEquipment
3. **判讀**：CapEx/營收比 >15% = 擴張期；<10% = 穩定期；負值 = 處分資產
4. **年報補充**：用 `pdf` 工具提取「在建工程明細」
5. **撰寫報告**：依下方回答模板結構產出

⚠️ **常見陷阱**：①答詢詳解 ≥200 字；②空頭視角 ≥2 個關鍵字；③診斷評級含數據

## 具體化思考點 (智能體強制執行指令)
- **【業績含金量審核】**：智慧體必須判定「產能封頂」的意義。若利用率長期 > 90% 且 CIP 持續投入，這不是風險，是「獲利即將噴發」的訊號；反之，若利用率下降但資產上升，必須警示「業績倒退風險」。
- **【未來動能指標】**：重點監控「在建工程 (CIP)」。將其視為「未來的產量儲備」。若 CIP 龐大且轉固在即，應視為營收即將躍遷的領先指標。
- **【輕資產化轉型識別】**：檢查「固定資產原值」是否有下降趨勢同時「毛利率」上升。若有，判定為「成功向輕資產/高毛利轉型」，屬重大利多。
- **【Ticker 校準】**：使用 yfinance 時，必須遵循 `.TW` (上市) 或 `.TWO` (上櫃) 規範，專門處理美股/ADR 時則直接輸入代碼。
- **【描述約束】**：嚴禁僅列出數字。必須給出「投資定性」——是處於「播種期 (CapEx 高)」、「收割期 (產能全開)」還是「老化期 (無產能更新)」。

## 回答模板與注意事項
#### 【F08 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：須採「業績定性+未來動向」結構（例：該公司目前產能封頂，在建工程金額龐大且轉固在即，判定業績處於 [爆發前夕] ，未來一年擬轉向 [高度成長] 軌道，資產利用效率極佳）。
- **【診斷評級】評分準則**：
    - **S (優)**：生意極好，利用率封頂，擴產精準且與現金流同步。
    - **A (良)**：穩定擴張，產能釋放與市場需求節奏對位。
    - **B (持平)**：產能過剩或擴產緩慢，獲利動能受限於既有資產效率。
- **【答詢詳解】解析深度**：依照以下三個維度回答：
    1. **生意火熱度 (Business Momentum)**：從利用率與營收增速讀懂市場需求。
    2. **擴產價值 (Expansion Value)**：CIP 的轉轉化潛力與對未來 EPS 的貢獻。
    3. **投資安全性 (Safety Margin)**：重資產架構下的防禦力與轉型效率。
- **【證據來源】數據溯源**：必須列出支撐「業績大好」或「轉型成功」的具體數據與來源。
- **【專家評析】核心分析與洞察內容**：分析「在建工程」轉「固定資產」的時間點是否與營收跳升脫節。若產能大幅開出但營收成長低於 10%，專家應在此處診斷為「無效擴張」或「資產減值」之先兆。判定該標的是「成長股之星 (擴產收割期)」還是「過氣巨頭 (資產老化)」。
