# F02: 營業現金流與獲利品質

- **本題 ID：** Q-F02
- **題目：** 營業活動現金流量 (CFO) 與稅後淨利之關係為何？獲利是否具備實質現金支撐？
- **推薦專家：** 財務分析師 (`finance-financial-analyst`)
- **推薦理由：** 專精於「現金流透視 (Cash Flow Transparency)」與「盈餘品質審計」。能從現金流量表中拆解「折舊」、「應收存貨變動」對獲利的實質影響，識別「有獲利無現金」的倒閉風險指標。
- **意圖：** 執行「盈餘含金量與獲利真實性審計」。透過測算 CFO 與 Net Income 的比值，判定獲利中「紙上富貴」的比例，識別潛在的「激進會計認列」風險，評估企業獲利對實質營運週轉的貢獻度。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockCashFlowsStatement` 提取 `CashFlowsFromOperatingActivities` 與 `NetIncome`。
    2. **【外部驗證：優分析/財經媒體 (cnYES/MoneyDJ)】**：透過 Web Search 檢索「[公司名] 獲利品質」、「營業現金流對淨利比」。
    3. **【穿透校準：Annual Report (年報)】**：查閱 `現金流量表附註 -> 非現金調節項目`。獲取折舊與攤銷之具體構成。
- **邏輯：** 執行「獲利含金量指標測試 (CFO/Net Income)」——觀察數值是否大於 100%（通常製造業標準）；執行「自由現金流 (FCF) 穿透」——評估剩餘現金之分紅強度。
- **來源：** `FinMind (現金流量表/損益表)`, `年度報告 (非現金項目明細)`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **跑工具**：`python3 財務分析工具.py cashflow_quality [股號]` → 取得 CFO/NI 比值、FCF 數據
2. **判讀 CFO/NI**：比值 >100% = 獲利含金量高；<100% = 盈餘虛胖；負值 = 營運現金流出
3. **年報補充**：若有疑慮，用 `pdf` 工具指定頁碼提取「應收帳款帳齡表」確認款項可回收性
4. **撰寫報告**：依下方回答模板結構產出

## 具體化思考點 (智能體強制執行指令)
- **【獲利比值測算】**：智慧體**必須**執行自最新年度 $Y$ 溯回至 $Y-3$ 之四年度財務對比分析，計算各年度的「CFO / 稅後淨利 (Net Income)」比值。
- **【非現金項目核實】**：若比值低於 100% 或持續下滑，必須深挖現金流量表中的「應收帳款增加」與「存貨增加」數值，判定獲利是否卡在倉庫或應收單據中。
- **【折舊掩蓋診斷】**：分析獲利是否僅靠「折舊費用 (Non-cash)」撐起，若扣除折舊後現金流轉負，必須標註為「資本高度密集但周轉效率低下」之警訊。
- **【Ticker 校準】**：使用 yfinance 時，必須遵循 `.TW` (上市) 或 `.TWO` (上櫃) 規範，嚴禁輸入純數字。
- **【描述約束】**：絕對禁用「獲利穩定」、「現金充足」。必須量化為「現金流對淨利比 %」與「自由現金流總額」。

## 回答模板與注意事項
#### 【F02 專家答詢】財務分析師 (finance-financial-analyst)
- **【專家直斷】品質要求**：須採「現金含金量+認列品質」結構（例：該公司 CFO 對淨利比長期高於 120%，顯示 [获利含金量] 極高；營運資金增加可控，[會計認列] 屬穩健範疇）。
- **【診斷評級】評分準則**：
    - **S (優)**：CFO 長期 > 淨利，且 FCF 同時為正，顯示公司之提款能力極強。
    - **A (良)**：CFO 穩定且與淨利長期同步，無明顯背離跡象。
    - **B (持平)**：CFO 連續數季/數年低於淨利，或 FCF 持續為負，顯示獲利品質正在惡化，具備營運資金斷鏈風險。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須啟用「華爾街空頭視角（Skepticism）」，強制演練最極端之風險阻力，並給出物理對策。
- **【證據來源】數據溯源**：必須精確列出 FinMind `cashflow` 指令回傳之數值及其計算過程。
