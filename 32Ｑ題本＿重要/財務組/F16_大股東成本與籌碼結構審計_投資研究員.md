# F16: 大股東成本與籌碼結構審計

- **本題 ID：** Q-F16
- **題目：** 董監事與大股東持股佔比與增減趨勢為何？董監持股質押比例是否過高？大股東的長期持股成本估值為何？
- **推薦專家：** 投資研究員 (`finance-investment-researcher`)
- **推薦理由：** 專精於「結構性籌碼分析」與「治理風險判讀」。能從股權分散表與質押變化中讀出大股東的「信心水位」與「財務壓力」，是判定股價下檔支撐與治理紅線的權威。
- **意圖：** 執行「利益綑綁與籌碼安全審計」。分析公司核心權力層的持股水位，判定大股東與小股東的利益是否一致。透過質押比例監控，識別潛在的「斷頭平倉」風險與「大股東財務危機」警訊。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心數據基石：FinMind】**：調用 `TaiwanStockInstitutionalHolding` 獲取董監持股與外資持股；調用 `TaiwanStockShareholdingSpread` 獲取大股東持股分級。
    2. **【關鍵指標檢索：優分析/財經媒體 (cnYES/MOPS)】**：獲取最新之「董監持股質押比例」趨勢。
    3. **【穿透校準：Annual Report (年報)】**：查閱「主要股東名單」及其相互關係。
- **邏輯：** 執行「質押安全性壓力測試」——若質押率 > 50% 且股價回檔 30%，判定具備「平倉連鎖反應」風險；執行「成本錨定算則」——分析大股東增減持點位以估算其平均成本區間。
- **來源：** `FinMind`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **跑工具**：`python3 財務分析工具.py valuation [股號]` → 取得 PER、股利
2. **取股權**：`python3 數據採集器.py backer_holding [股號]` → 抓大股東持股
3. **判讀**：董監質押 >30% = 斷頭風險；大股東減持 = 警訊
4. **年報補充**：用 `pdf` 工具從年報提取「主要股東名單」
5. **撰寫報告**：依下方回答模板結構產出

⚠️ **常見陷阱**：①答詢詳解 ≥200 字、專家評析 ≥200 字；②空頭視角 ≥2 個（股權集中、流動性差是重要利空）

## 具體化思考點 (智能體強制執行指令)
- **【籌碼集中度確認】**：智慧體**必須**列出 400 張與 1000 張大戶持股比例。嚴禁在未列出具體分佈前 conclud 「籌碼穩定」。
- **【質押爆雷特徵偵測】**：若「董監事持股質押比例」超過 30%，智慧體必須強制發出「財務槓桿過高」風險警訊，並標註目前的「平倉警戒價位（估值）」。
- **【利益綑綁度分析】**：分析近期董監事是否有「減持」行為。若業績看好但內部人持續出脫，必須判定為「內部人看淡前景」或「治理誠信疑慮」。
- **【描述指標】**：絕對禁用「持股大增」、「質押尚可」。必須量化為具體的「董監持股 %」、「質押率 %」與「千張大戶週變動 bps」。

## 回答模板與注意事項
#### 【F16 專家答詢】投資研究員 (finance-investment-researcher)
- **【專家直斷】品質要求**：須採「持股結構+信心判定」結構（例：大股東持股佔比 X%，質押率 Y%，屬 [利益深度綑綁/撤資外逃] 型態，籌碼支撐力道為 [強勁/渙散]）。
- **【診斷評級】評分準則**：
    - **S (優)**：董監持股高於 25% 且無質押，大戶持股穩定攀升，內部人近期有增持動作，展現極強的營運信心。
    - **A (良)**：持股比例穩定，無異常質押與減持，籌碼分佈健康，具備法人護盤基礎。
    - **B (持平)**：董監質押比例過高（> 50%）或大股東持續派發籌碼給散戶（散戶人數增、大戶減），顯示治理核心極度不穩且具備高度財務崩潰風險。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須採「多空雙向」結構——同時提供空方（利空風險）與多方（機會/利多）觀點，每方至少 2 點具體事實，最後綜合權衡。
- **【證據來源】數據溯源**：必須精確標註 FinMind `spread` 指令日期與質押百分比來源。
