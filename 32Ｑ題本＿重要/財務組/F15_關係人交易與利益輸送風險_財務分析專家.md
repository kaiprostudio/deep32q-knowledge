# F15: 關係人交易與利益輸送風險

- **本題 ID：** Q-F15
- **題目：** 是否存在異常大額的關係人交易（連屬公司、子公司）？其定價是否符合市場行情？是否存在利益輸送或虛增利潤之嫌？
- **推薦專家：** 財務分析專家 (`finance-financial-analyst`)
- **推薦理由：** 專精於「審計與公司治理」及「舞弊偵測」。具備從分部資訊（Segment Information）中識別異常交易流向的能力，能穿透複雜的交叉持股關係，判定資金往來是否具備「商業實質」，是守護股東權益的防線。
- **意圖：** 執行「關聯交易與治理透明度審計」。判定公司是否透過子公司進行「利潤調節」或「負債隱藏」，識別不具備公平市場定價之交易，預警潛在的利益輸送風險。
- **數據路徑與優先權 (Data Source Waterfall)：**
    1. **【核心事實來源：Annual Report (年報)】**：必須查閱「財務報告附註 -> 關係人交易」章節。詳列所有應收/應付關係人款項及銷貨/進貨明細。
    2. **【外部驗證：Web Search】**：檢索「[公司名] 子公司 掏空」、「[公司名] 關係人 訴訟」。
    3. **【穿透校準：MOPS】**：查閱「兩岸投資資訊」或「海外子公司營運狀況」，比對子公司之利潤率是否異常高於母公司。
- **邏輯：** 執行「利潤分佈背離測試」——若虧損的子公司仍有大量採購或母公司給予極長帳期，判定為「利益輸送」或「資產減損掩飾」；執行「定價公平性分析」。
- **來源：** `FinMind`

## 🛠️ 工作流程（按順序執行，不得跳過）

1. **年報提取**：用 `pdf` 工具從年報提取「關係人交易附註」
2. **分析**：關係人銷貨佔比、關係人應收款項、資金貸與、背書保證
3. **判讀**：關係人交易佔營收 >5% 需警惕；>10% = 利益輸送風險
4. **撰寫報告**：依下方回答模板結構產出

⚠️ **常見陷阱**：①答詢詳解 ≥200 字、專家評析 ≥200 字；②空頭視角 ≥2 個（股權集中、獨董不足是重要利空）

## 具體化思考點 (智能體強制執行指令)
- **【交易規模實測】**：智慧體**必須**列出「關係人交易佔營收/成本之百分比」。若比例超過 20% 且無合理解釋，必須標註為「高治理風險」。
- **【帳期異常偵測】**：比對關係人應收款項與一般客戶之週轉率。若關係人收款天數顯著拉長（例如超過 180 天），必須判定為「資金占用風險」。
- **【子公司利潤滲透】**：分析子公司是否淪為母公司的「垃圾桶（承接不良資產）」或「小金庫（虛增利潤）」。
- **【描述指標】**：絕對禁用「往來正常」、「合作緊密」。必須量化為具體的「關係人應收款金額」、「與關係人往來之毛利率階差」與「分部獲利貢獻比」。

## 回答模板與注意事項
#### 【F15 專家答詢】財務分析專家 (finance-financial-analyst)
- **【專家直斷】品質要求**：須採「交易佔比+定價公平性判定」結構（例：關係人進貨佔 X%，帳期較一般供應商 [拉長/縮短]，定價 [符合/偏離] 行情，治理風險等級為 [低/中/高]）。
- **【診斷評級】評分準則**：
    - **S (優)**：關係人交易比例極低（< 5%），且往來條件完全對齊市場第三方，透明度高。
    - **A (良)**：具備產業鏈整合之深度的關係人往來，但定價機制透明且經外部稽核確認具備商業合理性。
    - **B (持平)**：高度依賴特定關係人進行採購或銷貨，且往來餘額持續攀升，缺乏公平定價證據，具備高度利益輸送與財報粉飾風險。
    - **【答詢詳解】解析深度**：必須嚴格遵守 Fact-First 原則，確保每一段論述都有物理數字實體支撐，嚴禁脫離數據空泛推論。
    - **【專家評析】核心洞察**：必須採「多空雙向」結構——同時提供空方（利空風險）與多方（機會/利多）觀點，每方至少 2 點具體事實，最後綜合權衡。
- **【證據來源】數據溯源**：必須精確標註年報附註 P.XXX 之關係人交易彙總表。
